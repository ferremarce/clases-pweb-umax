In. FUNDAMENTACIÓN DEL PROGRAMA DE LA ASIGNATURA

En la era digital actual, el desarrollo web se ha consolidado como una de las competencias
más demandadas tanto en el ámbito profesional como en el académico. Las tecnologías de
la información y la comunicación (TIC) han transformado la manera en que las personas
interactúan, acceden a la información y realizan transacciones, lo que ha impulsado la
necesidad de contar con profesionales capacitados en el diseño, desarrollo y mantenimiento

de aplicaciones web.

La asignatura Introducción al Desarrollo Web tiene como propósito fundamental
proporcionar a los estudiantes los conocimientos esenciales para comprender el
funcionamiento de la web y las tecnologías asociadas al desarrollo de sitios y aplicaciones
web. A través de esta materia, los estudiantes adquirirán habilidades prácticas para crear
interfaces web interactivas, estructurar contenido utilizando HTML, dar estilo a través de

CSS y añadir funcionalidades dinámicas con JavaScript.

Además, esta asignatura promueve el desarrollo de competencias transversales, como el
pensamiento lógico, la resolución de problemas y el trabajo colaborativo, aspectos
fundamentales para el trabajo en equipo dentro de proyectos de desarrollo. De igual manera,
se abordarán aspectos éticos relacionados con la accesibilidad, la usabilidad y el respeto a
los estándares internacionales en la creación de soluciones web, alineados con las mejores

prácticas de la industria.

En un entorno donde la web evoluciona rápidamente y los avances tecnológicos surgen de

manera continua, es esencial que los estudiantes adquieran una base sólida en las

HI. COMPETENCIAS
a. Del perfil de salida

Diseñar, proyectar y programar soluciones informáticas, utilizando software, procesos u
otros medios tecnológicos, con el fin de desarrollar sistemas eficientes y adaptados a las

necesidades del entorno profesional.
b. Específica de la asignatura

Al finalizar la asignatura, el estudiante será capaz de erear sitios web interactivos utilizando
HTML, CSS y JavaScript, aplicando principios básicos de accesibilidad, usabilidad y

estándares web.

funcionalidad dinámica a las páginas web, como validaciones de formularios y

estilos en respuesta a las acciones del usuario.

Construye diseños web responsivos con Bootstrap, utilizando el sistema de rejilla de
Bootstrap para crear layouts que se adapten a diferentes tamaños de pantalla.
Además, integra componentes de Bootstrap como botones, menús de navegación,
modales y carruseles para mejorar la interfaz de usuario.

Publica páginas web en un servidor, subiendo un sitio web a un servidor usando FTP
o plataformas de hospedaje web. Además, configura GitHub Pages para la
publicación de sitios estáticos y utilizar sistemas de control de versiones para la
gestión de proyectos web.

ACTITUDINALES

Desarrolla una actitud proactiva hacia el aprendizaje continuo, manteniéndose
actualizado sobre las últimas tendencias y herramientas en el desarrollo web.
Además, participando en comunidades de desarrollo web para compartir
conocimientos y aprender de otros.

Fomenta la creatividad y la innovación en el diseño web, explorando nuevas formas
de mejorar la experiencia de usuario en las páginas web, buscando soluciones
originales. Además, valora el diseño web accesible e inclusivo, respetando las
necesidades de todos los usuarios.

Adopta una actitud ética y responsable en la publicación de contenido, respetando los
derechos de autor y las licencias de los recursos utilizados (imágenes, fuentes, etc.).
Además, asegura que los sitios web publicados sean accesibles y seguros,
protegiendo la privacidad y los datos de los usuarios.
