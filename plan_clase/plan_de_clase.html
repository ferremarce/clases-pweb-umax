📋 Estructura de las 18 Sesiones

El siguiente plan de clase está alineado con las unidades programáticas del sílabo, integrando teoría, práctica y el proyecto final. Cada sesión combina exposición de conceptos, ejercicios guiados y trabajo colaborativo.

📌 Recomendación sobre el Proyecto Final (Unidad VI)

El proyecto final se desarrolla en las sesiones 15 a 18. Se sugiere dedicar la sesión 15 exclusivamente a la planificación, creación de repositorios y organización de tareas. Las sesiones 16 y 17 se centran en la implementación técnica (HTML/CSS y JavaScript), mientras que la sesión 18 combina publicación en GitHub Pages con presentaciones orales. Si el número de grupos es elevado, considere ampliar la evaluación a una sesión extra o solicitar entregas parciales.

Evaluación sugerida: Rúbrica que valore funcionamiento técnico, uso de estándares, creatividad, trabajo colaborativo (evidenciado en Git) y calidad de la presentación.

💡 Propuesta de Proyectos para Grupos

Cada grupo (3 a 5 integrantes) seleccionará una de las siguientes ideas o propondrá una similar, con aprobación del docente. Los proyectos deben evidenciar el dominio de HTML, CSS, JavaScript y Bootstrap.

📁 Portafolio Personal Avanzado

Galería de proyectos filtrable con JS, formulario de contacto validado, layout con Grid/Flexbox y barra de navegación responsiva con Bootstrap.

HTML semántico · CSS Grid · Bootstrap · JS (filtros)

✈️ Guía de Viajes / Blog de Turismo

Inicio con tarjetas de destinos, página detallada, tabla de información práctica, suscripción a newsletter y mapa interactivo (iframe).

Multimedia · Formularios · Tablas · JS para mostrar/ocultar contenido

🛒 Tienda de Música / E-commerce Simple

Catálogo de productos, carrito de compras con localStorage, detalle de producto y formulario de checkout con validación avanzada.

Grid de Bootstrap · Manipulación del DOM · localStorage · Eventos

🎮 Club de Fans / Comunidad

Noticias con cards, galería de imágenes, foro estático, formulario de registro con validación y modales de Bootstrap.

Bootstrap components · JS validación · Pestañas dinámicas · CSS personalizado

🍳 Página de Recetas de Cocina

Grid de recetas destacadas, página detallada con lista ingredientes, tabla nutricional, buscador en JS que filtre recetas.

Listas, tablas, Flexbox · JavaScript (filtro) · Formulario envío recetas

📌 Metodología General por Sesión

Cada sesión de 4 horas se estructura de la siguiente manera:

  • Teoría (1 - 1.5h): Exposición de conceptos clave con ejemplos prácticos y referencias a la bibliografía.
  • Práctica guiada (2 - 2.5h): Ejercicios en clase, desarrollo incremental de proyectos y resolución de dudas.
  • Trabajo independiente / cierre (0.5h): Asignación de tareas cortas, lecturas complementarias o avance del proyecto.

El proyecto final integra todas las unidades, permitiendo evaluar las competencias específicas: crear sitios web interactivos con HTML, CSS y JS aplicando accesibilidad, usabilidad y estándares web.
