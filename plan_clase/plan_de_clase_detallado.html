Introducción al Desarrollo Web | 18 sesiones de 4 horas
📅 18 sesiones ⏱️ 4 horas / sesión 🎓 Total: 72 horas presenciales 💻 Enfoque teórico-práctico

📖 Desarrollo detallado de las 18 sesiones

Cada sesión de 4 horas está estructurada para maximizar el aprendizaje combinando fundamentos, ejemplos en vivo, ejercicios prácticos y avance del proyecto final. A continuación se despliega el contenido específico por sesión en formato de viñetas.

Sesión 1 Fundamentos de la Web y Entorno de Trabajo 📝 Ejercicios Unidad I
  • Conceptos básicos: ¿Qué es Internet? Diferencia entre web, servidor, cliente, navegador.
  • Historia breve de la web: Web 1.0, 2.0, estándares W3C.
  • Instalación y configuración de Visual Studio Code (extensiones: Live Server, Prettier, Auto Rename Tag).
  • Estructura básica de un documento HTML: doctype, html, head, body.
  • Primera página web: etiquetas de título, párrafos, encabezados h1-h6.
  • Uso de Live Server para visualización en tiempo real.
  • Introducción a rutas de archivos y organización de carpetas para proyectos web.

📌 Actividades (4h)

Teoría 1h Exposición interactiva + navegación por sitios reales.
Práctica 2.5h Configuración de VS Code, creación de "Mi primera página" con título, biografía personal y lista de hobbies. Uso de Live Server.
Cierre 0.5h Revisión del código de los ejercicios, presentación del funcionamiento y discusión de lecciones aprendidas. Tarea: investigar sobre servidores web y dominios.

Sesión 2 HTML I: Estructura, Texto y Listas 📝 Ejercicios Unidad II
  • Semántica HTML: etiquetas estructurales (header, nav, main, section, article, footer).
  • Formato de texto: strong, em, mark, small, blockquote, cite.
  • Listas ordenadas (ol), no ordenadas (ul) y listas de definición (dl).
  • Anidación correcta de listas y buenas prácticas de indentación.
  • Validación del código con W3C Validator.
  • Comentarios en HTML y organización de código.

📌 Actividades (4h)

Teoría 1.2h Explicación de semántica y ejemplos comparativos.
Práctica 2.5h Construir página personal con estructura semántica, biografía, lista de habilidades y pasatiempos con listas anidadas.
Cierre 0.3h Revisión del código de los ejercicios, presentación del funcionamiento y discusión de lecciones aprendidas.

Sesión 3 HTML II: Enlaces, Imágenes y Multimedia 📝 Ejercicios Unidad II
  • Enlaces: rutas absolutas vs relativas, atributo target (_blank, _self).
  • Enlaces internos (anclas) y navegación entre páginas.
  • Etiqueta img: atributos src, alt, width, height. Buenas prácticas de accesibilidad.
  • Formatos de imagen modernos (webp, svg).
  • Inserción de audio y video con <audio> y <video> (controls, autoplay, loop).
  • Iframes: incrustar mapas, videos de YouTube.
  • Creación de una barra de navegación básica con enlaces.

📌 Actividades (4h)

Teoría 1h Demostración de rutas, multimedia.
Práctica 2.5h Ampliar la página personal: agregar galería de imágenes, video incrustado, menú de navegación que conecte secciones internas y una página adicional "acerca de".
Cierre 0.5h Revisión del código de los ejercicios, presentación del funcionamiento y discusión de lecciones aprendidas.

Sesión 4 HTML III: Tablas y Formularios 📝 Ejercicios Unidad II
  • Tablas: table, tr, th, td, thead, tbody, tfoot, colspan y rowspan.
  • Atributos de accesibilidad en tablas (scope).
  • Formularios: <form>, action, method GET/POST.
  • Campos: input (text, email, password, number, date, radio, checkbox), textarea, select, button.
  • Atributos de validación: required, minlength, pattern, placeholder.
  • Agrupación con fieldset y legend.
  • Buenas prácticas en formularios para UX.

📌 Actividades (4h)

Teoría 1.2h Explicación de formularios, métodos HTTP.
Práctica 2.5h Crear tabla comparativa de lenguajes web. Diseñar formulario de contacto completo con validaciones HTML5 (sin JS aún).
Cierre 0.3h Revisión del código de los ejercicios, presentación del funcionamiento y discusión de lecciones aprendidas.

Sesión 5 CSS I: Fundamentos, Color y Texto 📝 Ejercicios Unidad III
  • Introducción a CSS: sintaxis, selectores por etiqueta, clase, ID, combinadores básicos.
  • Formas de aplicar CSS: inline, interno, externo (mejor práctica).
  • Colores: nombres, hexadecimal, rgb, hsl, rgba.
  • Propiedades de texto: font-family, font-size, font-weight, text-align, line-height.
  • Modelo de caja introductorio: content, padding, border, margin.
  • Unidades de medida: px, em, rem, %.

📌 Actividades (4h)

Teoría 1h Explicación de cascada, especificidad.
Práctica 2.5h Tomar página personal de sesiones anteriores, crear archivo CSS externo, aplicar estilos tipográficos, colores corporativos, diseño de tarjetas.
Cierre 0.5h Revisión del código de los ejercicios, presentación del funcionamiento y discusión de lecciones aprendidas.

Sesión 6 CSS II: Modelo de Cajas y Layouts Básicos 📝 Ejercicios Unidad III
  • Profundización en modelo de caja: box-sizing: border-box.
  • Display: block, inline, inline-block, none.
  • Posicionamiento: static, relative, absolute, fixed, sticky.
  • Propiedades z-index, overflow.
  • Introducción a floats y limpieza con clearfix.
  • Diseño de columnas básicas con float.

📌 Actividades (4h)

Teoría 1.2h Demostración de posicionamiento y floats.
Práctica 2.5h Maquetar un layout de dos columnas usando float. Crear header fijo sticky, sidebar flotante. Ajustar márgenes y paddings.
Cierre 0.3h Revisión del código de los ejercicios, presentación del funcionamiento y discusión de lecciones aprendidas.

Sesión 7 CSS III: Flexbox y CSS Grid 📝 Ejercicios Unidad III
  • Flexbox: container (display:flex), flex-direction, justify-content, align-items, flex-wrap.
  • Propiedades para items: order, flex-grow, flex-shrink.
  • CSS Grid: grid-template-columns, grid-template-rows, gap, áreas de plantilla.
  • Comparativa entre Flexbox y Grid: cuándo usar cada uno.
  • Diseño responsivo con unidades fr, auto.

📌 Actividades (4h)

Teoría 1.5h Ejemplos interactivos con Flexbox Froggy y Grid Garden.
Práctica 2.2h Reconstruir el layout de la página personal usando Flexbox para navbar y Grid para galería de proyectos.
Cierre 0.3h Revisión del código de los ejercicios, presentación del funcionamiento y discusión de lecciones aprendidas.

Sesión 8 CSS IV: Formularios, Tablas y Diseño Responsivo 📝 Ejercicios Unidad III
  • Estilización de formularios: inputs, botones, focus, validaciones visuales.
  • Estilos para tablas: zebra striping, hover, bordes.
  • Media queries: breakpoints comunes, diseño mobile-first.
  • Uso de viewport meta tag.
  • Layout responsivo con Flexbox/Grid.
  • Imágenes flexibles: max-width:100%.

📌 Actividades (4h)

Teoría 1h Media queries, ejemplos de sitios adaptables.
Práctica 2.5h Tomar el formulario de sesión 4 y estilizarlo con CSS avanzado. Adaptar toda la página personal para verse correctamente en móvil, tablet y escritorio.
Cierre 0.5h Revisión del código de los ejercicios, presentación del funcionamiento y discusión de lecciones aprendidas.

Sesión 9 JavaScript I: Variables, Tipos y Operadores 📝 Ejercicios Unidad IV
  • Introducción: qué es JS, inserción en HTML (interno/externo).
  • Declaración de variables: var, let, const, ámbito.
  • Tipos primitivos: string, number, boolean, null, undefined.
  • Operadores aritméticos, asignación, comparación (== vs ===).
  • Interacción básica: alert, prompt, console.log.
  • Conversión de tipos (coerción).

📌 Actividades (4h)

Teoría 1.2h Conceptos fundamentales con ejemplos en vivo.
Práctica 2.5h Ejercicios prácticos: calculadora básica, conversor de temperatura, uso de prompt.
Cierre 0.3h Revisión del código de los ejercicios, presentación del funcionamiento y discusión de lecciones aprendidas.

Sesión 10 JavaScript II: Condicionales e Iteraciones 📝 Ejercicios Unidad IV
  • Estructuras condicionales: if/else, else if, switch.
  • Operador ternario.
  • Bucles: for, while, do while.
  • Iterar arrays con forEach (introducción).
  • Ejercicios de lógica: validación de formularios simple, tabla de multiplicar.

📌 Actividades (4h)

Teoría 1.2h Explicación de condicionales y bucles con ejemplos.
Práctica 2.5h Desarrollar juego de adivinanza, simulación de notas, validación de contraseña con bucle. Práctica en parejas.
Cierre 0.3h Revisión del código de los ejercicios, presentación del funcionamiento y discusión de lecciones aprendidas.

Sesión 11 JavaScript III: Funciones y Eventos del DOM 📝 Ejercicios Unidad IV
  • Declaración de funciones: expresión, declaración, arrow functions.
  • Parámetros y retorno.
  • Introducción al DOM: document, selección de elementos (getElementById, querySelector).
  • Eventos: click, mouseover, submit, change. addEventListener.
  • Modificar contenido y estilos desde JS.

📌 Actividades (4h)

Teoría 1.2h Explicación de funciones y eventos del DOM.
Práctica 2.5h Interacción: botón que cambia color de fondo, mostrar/ocultar texto. Integrar eventos en formulario de contacto.
Cierre 0.3h Revisión del código de los ejercicios, presentación del funcionamiento y discusión de lecciones aprendidas.

Sesión 12 JavaScript IV: Manipulación DOM y Almacenamiento Unidad IV
  • Crear y eliminar elementos dinámicamente (createElement, appendChild).
  • localStorage y sessionStorage: guardar, recuperar datos.
  • Proyecto práctico: Lista de tareas con agregar, eliminar y persistencia.
  • Eventos de teclado y formularios avanzados.

📌 Recomendaciones metodológicas

Para un correcto aprovechamiento de las 4 horas, se recomienda fraccionar cada sesión en bloques de 50-60 minutos con pausas activas. Se sugiere utilizar plataformas colaborativas (GitHub Classroom, Google Drive) y realizar revisiones semanales de avance. El proyecto final debe ser incremental, con entregas parciales en las sesiones 15, 16 y 17 para retroalimentación temprana.
