Proyecto Final 1: Tienda Online

INI04 - Introduccion al Desarrollo Web

Prof. Ing. Juan M. Ferreira

Descripcion General

Desarrollar una tienda online funcional que muestre un catalogo de productos organizados por categorias, con un carrito de compras interactivo y un formulario de checkout. El proyecto integra todos los conocimientos adquiridos en las sesiones 1 a 12: HTML semantico, CSS responsivo y JavaScript con manipulacion del DOM.

Duracion estimada: 3-4 sesiones de clase
Modalidad: Grupos de 3 estudiantes
Tecnologias: HTML5, CSS3, JavaScript (ES6+)

Requisitos HTML

  1. Estructura semantica completa: <header>, <nav>, <main>, <section>, <footer>
  2. Barra de navegacion fija con enlaces internos (anclas) a cada seccion y un contador del carrito
  3. Seccion de catalogo con tarjetas de producto (imagen, nombre, precio, descripcion, boton "Agregar al carrito")
  4. Seccion de filtros por categoria con botones para seleccionar Tecnologia, Ropa o Libros
  5. Seccion de carrito con tabla semantica (<thead>, <tbody>, <tfoot>) que muestre productos agregados
  6. Formulario de checkout con campos: nombre completo, email, telefono, direccion, ciudad
  7. Validacion HTML5 en el formulario: required, type="email", minlength, pattern
  8. Uso de <fieldset> y <legend> para agrupar campos del formulario
  9. Footer con nombres de los integrantes del equipo
  10. Atributo alt descriptivo en todas las imagenes

Requisitos CSS

  1. Variables CSS (:root) para la paleta de colores del proyecto
  2. CSS Grid para el catalogo de productos: 1 columna en movil, 2 en tablet, 3-4 en escritorio
  3. Flexbox para la navbar, tarjetas de producto y layout del carrito
  4. Diseno mobile-first con media queries (breakpoints: 576px, 768px, 992px)
  5. Estados interactivos: :hover en tarjetas y botones, :focus en inputs
  6. Badge de categoria en cada tarjeta usando position: absolute sobre contenedor con position: relative
  7. Transiciones suaves (transition) en botones y tarjetas
  8. Tabla del carrito con zebra striping (nth-child) y hover en filas
  9. Formulario estilizado con estados :valid e :invalid
  10. Navbar fija con position: fixed y z-index

Requisitos JavaScript

  1. Funcion que genera las tarjetas del catalogo dinamicamente con createElement
  2. Funcion que filtra y re-renderiza productos segun la categoria seleccionada
  3. Funcion que agrega un producto al array del carrito
  4. Funcion que elimina un producto del carrito
  5. Funcion que re-renderiza la tabla del carrito y recalcula el total
  6. Funcion que retorna el total de la compra (suma de precio x cantidad)
  7. Funcion que valida los campos del formulario antes de confirmar la compra
  8. Todos los eventos deben registrarse con addEventListener (no usar onclick inline)
  9. Usar const y let (no var), arrow functions cuando corresponda, y template literals
  10. Los datos del carrito se manejan en memoria (array), no se requiere localStorage

Resultado Esperado

  • La pagina carga mostrando los 12 productos organizados en un grid responsivo
  • Al hacer clic en un boton de categoria, se filtran y muestran solo los productos de esa categoria
  • Al hacer clic en "Agregar al carrito", el producto aparece en la tabla del carrito y el contador de la navbar se actualiza
  • Se puede eliminar productos del carrito y el total se recalcula automaticamente
  • El formulario de checkout valida los datos antes de mostrar un mensaje de confirmacion
  • La pagina es completamente responsiva: se adapta a movil, tablet y escritorio
  • El codigo esta organizado, indentado y sigue las convenciones del curso (camelCase en JS, snake_case en CSS)

Entrega

  1. Formato: Carpeta comprimida (.zip) con la estructura de archivos indicada
  2. Estructura minima de archivos:
    • index.html - Pagina principal
    • css/estilos.css - Hoja de estilos
    • js/app.js - Logica de la aplicacion
    • images/ - Carpeta de imagenes (si aplica)
  3. Defensa oral: Cada integrante debe explicar una parte del proyecto y responder preguntas del docente
  4. Fecha limite: A definir por el docente
