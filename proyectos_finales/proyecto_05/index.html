Proyecto Final 5: Quiz Interactivo Educativo

INI04 - Introduccion al Desarrollo Web

Prof. Ing. Juan M. Ferreira

Descripcion General

Desarrollar una plataforma de cuestionarios interactivos con multiples categorias, temporizador por pregunta, puntuacion dinamica y retroalimentacion inmediata. El proyecto integra HTML semantico, CSS responsivo y JavaScript con manipulacion del DOM.

Duracion estimada: 3-4 sesiones de clase
Modalidad: Grupos de 3 estudiantes
Tecnologias: HTML5, CSS3, JavaScript (ES6+)

Requisitos HTML

  1. Estructura semantica completa: <header>, <nav>, <main>, <section>, <footer>
  2. Tres pantallas/secciones principales: Inicio (seleccion de categoria), Quiz (preguntas), Resultados
  3. Pantalla de inicio con botones de categoria y descripcion del quiz
  4. Pantalla de quiz con: numero de pregunta actual, barra de progreso, pregunta, opciones (radio buttons), temporizador, botones de navegacion
  5. Pantalla de resultados con: puntuacion final, tabla semantica con historial de intentos, boton para reiniciar
  6. Navegacion entre pantallas (Inicio, Quiz, Resultados) con indicador de seccion activa
  7. Uso de <fieldset> y <legend> para agrupar opciones de respuesta
  8. Tabla de resultados con <thead>, <tbody>, <tfoot>
  9. Footer con nombres de los integrantes del equipo
  10. Atributos alt descriptivos en todas las imagenes

Requisitos CSS

  1. Variables CSS (:root) para la paleta de colores del proyecto
  2. Flexbox para centrado de contenido, botones y layout general
  3. Diseno mobile-first con media queries (breakpoints: 576px, 768px, 992px)
  4. Estados interactivos: :hover en botones y opciones, :checked en radio buttons
  5. Transiciones suaves (transition) en botones, opciones y cambios de pantalla
  6. Barra de progreso con ancho dinamico y color que cambia segun el progreso
  7. Colores de retroalimentacion: verde para respuesta correcta, rojo para incorrecta
  8. Tarjetas de categoria con position: relative y badges con position: absolute
  9. Temporizador con estilo destacado (tamano grande, color que cambia cuando queda poco tiempo)
  10. Animacion de entrada/salida al cambiar de pantalla usando display y opacity

Requisitos JavaScript

  1. Array de preguntas con estructura { id, categoria, pregunta, opciones: [], respuestaCorrecta: numero }, minimo 15 en 3 categorias
  2. Funcion que filtra preguntas por categoria e inicializa el estado del quiz
  3. Funcion que renderiza dinamicamente la pregunta actual con createElement
  4. Funcion que compara la respuesta seleccionada con la correcta y muestra retroalimentacion visual
  5. Funcion que avanza a la siguiente pregunta o muestra resultados si es la ultima
  6. Funciones que manejan el temporizador con setInterval (iniciar) y clearInterval (detener)
  7. Funcion que retorna el porcentaje de respuestas correctas
  8. Funcion que renderiza la pantalla de resultados con puntuacion y tabla de historial
  9. Funcion que actualiza el ancho de la barra segun la pregunta actual
  10. Buenas practicas: eventos con addEventListener, uso de const/let, arrow functions, template literals y datos en memoria sin localStorage

Resultado Esperado

  • La pagina carga mostrando la pantalla de inicio con botones de categorias
  • Al seleccionar una categoria, se inicia el quiz con la primera pregunta
  • La barra de progreso se actualiza con cada pregunta
  • El temporizador cuenta regresivamente desde 30 segundos por pregunta
  • Al seleccionar una respuesta, se muestra retroalimentacion inmediata (verde/rojo)
  • El boton "Siguiente" avanza a la siguiente pregunta o muestra resultados
  • Si el tiempo se agota, avanza automaticamente a la siguiente pregunta
  • La pantalla de resultados muestra: puntuacion final, porcentaje, tabla con historial de intentos
  • El boton "Reiniciar" permite volver a la pantalla de inicio
  • La pagina es completamente responsiva: se adapta a movil, tablet y escritorio
  • El codigo esta organizado, indentado y sigue las convenciones del curso (camelCase en JS, snake_case en CSS)

Entrega

  1. Formato: Carpeta comprimida (.zip) con la estructura de archivos indicada
  2. Estructura minima de archivos:
    • index.html - Pagina principal
    • css/estilos.css - Hoja de estilos
    • js/app.js - Logica de la aplicacion
    • images/ - Carpeta de imagenes (si aplica)
  3. Defensa oral: Cada integrante debe explicar una parte del proyecto y responder preguntas del docente
  4. Fecha limite: A definir por el docente
