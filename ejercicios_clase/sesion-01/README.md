# Ejercicios - Sesión 1: Fundamentos de la Web

## Objetivo
Practicar los conceptos básicos de HTML creando páginas web con estructura correcta y etiquetas fundamentales.

## Ejercicio 1: Mi Primera Página Personal
**Dificultad:** Básico | **Duración:** 45 minutos

Crea una página web personal básica con:
- Estructura HTML5 correcta
- Título principal (h1) con tu nombre
- Párrafos con información personal
- Al menos un subtítulo (h2)
- Etiquetas de formato: `<strong>`, `<em>`
- Una línea horizontal `<hr>`

**Archivos necesarios:**
- `index.html`

## Ejercicio 2: Explorando Etiquetas de Texto
**Dificultad:** Básico | **Duración:** 30 minutos

Practica diferentes etiquetas de texto:
- Encabezados h1 hasta h6
- Párrafos `<p>`
- Salto de línea `<br>`
- Línea horizontal `<hr>`
- Texto importante `<strong>`
- Texto enfatizado `<em>`

**Archivos necesarios:**
- `index.html`

## Ejercicio 3: Página de Presentación Completa
**Dificultad:** Intermedio | **Duración:** 1 hora

Crea una página más elaborada:
- Múltiples secciones usando encabezados
- Lista de habilidades/hobbies
- Información de contacto ficticia
- Uso correcto de todas las etiquetas vistas

**Archivos necesarios:**
- `index.html`

## Requisitos Generales
1. Todos los archivos deben tener extensión `.html`
2. Usar la estructura HTML5 con `<!DOCTYPE html>`
3. Incluir el atributo `lang="es"` en la etiqueta `<html>`
4. Agregar meta charset UTF-8
5. Agregar meta viewport para responsividad
6. Verificar que el código funcione en el navegador

## Herramientas Recomendadas
- Visual Studio Code como editor
- Extensión Live Server para preview en tiempo real
- W3C Validator para verificar HTML

## Entrega
Guarda todos los archivos en la carpeta `sesion-01` siguiendo la estructura:
```
sesion-01/
├── ejercicio-01/
│   └── index.html
├── ejercicio-02/
│   └── index.html
└── ejercicio-03/
    └── index.html
```
