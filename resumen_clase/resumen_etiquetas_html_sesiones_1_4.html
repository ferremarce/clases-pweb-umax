SESIÓN 1: Fundamentos de la Web y Entorno de Trabajo

Estructura del documento
<!DOCTYPE html>

Declara el tipo de documento (HTML5)

<!DOCTYPE html>
▶ Visualización:
No se muestra. Indica al navegador que renderice en modo estándar.
<html>

Elemento raíz del documento

<html lang="es">...</html>
▶ Visualización:
Contenedor invisible de toda la página.
<head>

Metadatos del documento (no visible)

<head> <meta charset="UTF-8"> <title>Mi Página</title> </head>
▶ Visualización:
No se muestra. Configura el documento.
<meta>

Metadatos (charset, viewport, descripción, autor)

<meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <meta name="description" content="Descripción para SEO">
▶ Visualización:
No visible. charset permite mostrar ñ y acentos; viewport habilita diseño responsive.
<title>

Título de la pestaña del navegador

<title>Mi Primera Página</title>
▶ Visualización:
Texto en la pestaña del navegador y en resultados de Google.
<link>

Enlace a recursos externos (CSS, favicon)

<link rel="stylesheet" href="css/style.css"> <link rel="icon" href="images/favicon.png">
▶ Visualización:
Aplica estilos CSS y muestra el icono en la pestaña.
<body>

Contenido visible de la página

<body> <h1>¡Hola Mundo!</h1> </body>
▶ Visualización:
Todo lo que el usuario ve está dentro de <body>.

Etiquetas de Texto

<h1> a <h6>

Encabezados (jerarquía)

<h1>Encabezado Principal</h1> <h2>Subtítulo</h2> <h3>Subsección</h3>
▶ Visualización:

Encabezado Principal

Subtítulo

Subsección

<p>

Párrafo

<p>Este es un párrafo de texto.</p>
▶ Visualización:

Este es un párrafo de texto.

<strong>

Texto importante (negrita semántica)

<p>Esto es <strong>muy importante</strong></p>
▶ Visualización:
Esto es muy importante
<em>

Énfasis (cursiva semántica)

<p>Esto es <em>enfatizado</em></p>
▶ Visualización:
Esto es enfatizado
<br>

Salto de línea

<p>Línea 1<br>Línea 2</p>
▶ Visualización:
Línea 1
Línea 2
<hr>

Línea horizontal

<p>Texto arriba</p> <hr> <p>Texto abajo</p>
▶ Visualización:

Texto arriba

Texto abajo

Etiquetas Semánticas

<header>

Encabezado de página o sección

<header> <h1>Mi Sitio Web</h1> </header>
▶ Visualización:
Zona superior de la página (requiere CSS para verse).
<main>

Contenido principal

<main> <p>Contenido principal único.</p> </main>
▶ Visualización:
Área central de contenido (requiere CSS).
<section>

Sección temática

<section> <h2>Sobre Mí</h2> <p>Soy estudiante...</p> </section>
▶ Visualización:
Bloque agrupado temáticamente (requiere CSS).
<nav>

Navegación

<nav> <a href="#inicio">Inicio</a> <a href="#contacto">Contacto</a> </nav>
▶ Visualización:
Inicio | Contacto
<figure> y <figcaption>

Contenido ilustrativo con leyenda

<figure> <img src="imagen.jpg" alt="Diagrama"> <figcaption>Descripción</figcaption> </figure>
▶ Visualización:

Listas

<ul>

Lista desordenada

<ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ul>
▶ Visualización:
  • HTML
  • CSS
  • JavaScript
<ol>

Lista ordenada

<ol> <li>Abrir VS Code</li> <li>Crear archivo HTML</li> <li>Abrir con Live Server</li> </ol>
▶ Visualización:
  1. Abrir VS Code
  2. Crear archivo HTML
  3. Abrir con Live Server
<li>

Elemento de lista

<li>Elemento individual</li>
▶ Visualización:
Un ítem con viñeta o número según el padre.
<img>

Imagen

<img src="imagen.jpg" alt="Descripción" width="200">
▶ Visualización:
<a>

Enlace / hipervínculo

<a href="pagina.html" target="_blank">Enlace</a>
▶ Visualización:
Enlace (azul y subrayado)

SESIÓN 2: HTML I — Estructura, Texto y Listas

Semánticas adicionales
<article>

Contenido independiente

<article> <h2>Título del artículo</h2> <p>Contenido...</p> </article>
▶ Visualización:
Bloque de contenido autocontenido (ej: post de blog).
<aside>

Contenido relacionado lateral

<aside> <h3>Relacionados</h3> <ul><li>Artículo 1</li></ul> </aside>
▶ Visualización:
Barra lateral con contenido complementario.
<footer>

Pie de página

<footer> <p>© 2024 Mi Sitio Web</p> </footer>
▶ Visualización:
Zona inferior de la página.

Formato de Texto

<u>

Subrayado

<p>Texto <u>subrayado</u></p>
▶ Visualización:
Texto subrayado
<s>

Tachado

<p>Texto <s>tachado</s></p>
▶ Visualización:
Texto tachado
<mark>

Texto resaltado

<p>Texto <mark>resaltado</mark></p>
▶ Visualización:
Texto resaltado (fondo amarillo)
<abbr>

Abreviatura

<abbr title="HyperText Markup Language">HTML</abbr>
▶ Visualización:
HTML (pasa el mouse)
<blockquote>

Cita larga

<blockquote cite="fuente"> <p>Una cita importante.</p> </blockquote>
▶ Visualización:
Una cita importante.
<cite>

Título de obra

<p>El libro <cite>El Gran Gatsby</cite>.</p>
▶ Visualización:
El libro El Gran Gatsby es un clásico.
<details> y <summary>

Contenido colapsable

<details> <summary>Haz clic para ver más</summary> <p>Contenido oculto.</p> </details>
▶ Visualización:
Haz clic para ver más

Contenido que aparece al hacer clic.

SESIÓN 3: HTML II — Enlaces, Imágenes y Multimedia

Multimedia
<audio>

Reproductor de audio

<audio controls> <source src="audio.mp3" type="audio/mpeg"> Tu navegador no soporta audio. </audio>
▶ Visualización:
Barra de reproducción con play/pausa, volumen.
<video>

Reproductor de video

<video controls width="320"> <source src="video.mp4" type="video/mp4"> </video>
▶ Visualización:
Reproductor de video con controles.
<source>

Fuente de multimedia

<source src="video.mp4" type="video/mp4">
▶ Visualización:
No visible por sí mismo. El navegador elige el formato compatible.
<iframe>

Contenido embebido externo (YouTube, Maps)

<iframe width="300" height="150" src="https://www.youtube.com/embed/VIDEO_ID" frameborder="0" allowfullscreen> </iframe>
▶ Visualización:
Ventana incrustada mostrando contenido externo.

SESIÓN 4: HTML III — Tablas y Formularios

Tablas
<table>

Contenedor de tabla

<table> <thead> <tr><th>Producto</th><th>Precio</th></tr> </thead> <tbody> <tr><td>Laptop</td><td>$999</td></tr> </tbody> </table>
▶ Visualización:
ProductoPrecio
Laptop$999
<thead>, <tbody>, <tfoot>

Secciones de la tabla

<thead>Encabezado</thead> <tbody>Cuerpo de datos</tbody> <tfoot>Pie/Totales</tfoot>
▶ Visualización:
thead: negrita con fondo oscuro | tbody: datos | tfoot: resumen final
<tr>

Fila de tabla

<tr> <td>Dato 1</td> <td>Dato 2</td> </tr>
▶ Visualización:
Una fila horizontal de celdas.
<th>

Celda de encabezado

<th scope="col">Producto</th>
▶ Visualización:
Producto (negrita y centrado)
<td>

Celda de datos

<td>Laptop</td>
▶ Visualización:
Laptop (texto normal de celda)
colspan / rowspan

Combinar celdas

<td colspan="2">Ocupa 2 columnas</td> <td rowspan="2">Ocupa 2 filas</td>
▶ Visualización:
Celda que se extiende horizontal o verticalmente.

Formularios

<form>

Contenedor de formulario

<form action="procesar.php" method="POST"> <!-- campos --> </form>
▶ Visualización:
Contenedor invisible. Envía datos vía GET o POST.
<label>

Etiqueta de campo

<label for="email">Email:</label> <input type="email" id="email">
▶ Visualización:
<input>

Campo de entrada (múltiples tipos)

<input type="text"> <input type="email"> <input type="password"> <input type="checkbox"> <input type="radio"> <input type="date"> <input type="number">
▶ Visualización:



checkbox radio
<textarea>

Área de texto multilínea

<textarea name="comentario" rows="4" cols="30"></textarea>
▶ Visualización:
<select> y <option>

Lista desplegable

<select name="pais"> <option value="py">Paraguay</option> <option value="ar">Argentina</option> </select>
▶ Visualización:
<button>

Botón

<button type="submit">Enviar</button> <button type="reset">Limpiar</button>
▶ Visualización:
<fieldset> y <legend>

Agrupación de campos

<fieldset> <legend>Información Personal</legend> <label>Nombre:</label> <input type="text"> </fieldset>
▶ Visualización:
Información Personal
