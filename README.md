## 🎯 Competencias

### Competencia del Perfil de Salida

Diseñar, proyectar y programar soluciones informáticas, utilizando software, procesos u otros medios tecnológicos, con el fin de desarrollar sistemas eficientes y adaptados a las necesidades del entorno profesional.

### Competencia Específica de la Asignatura

Al finalizar la asignatura, el estudiante será capaz de crear sitios web interactivos utilizando HTML, CSS y JavaScript, aplicando principios básicos de accesibilidad, usabilidad y estándares web.

### Unidad I: Introducción al Desarrollo Web (Sesión 1)

- La Web: significado, funcionamiento, navegadores web
- Desarrollo web: Definición, tipos, herramientas, lenguajes, formatos
- Reglas generales para el desarrollo web:
  - Legibilidad
  - Posicionamiento
  - Carpetas y rutas de archivos
  - Editores de Código

### Unidad II: HTML Básico (Sesiones 2-4)

- Introducción al HTML
- Versiones de HTML
- Textos
- Listas
- Enlaces
- Imágenes
- Audio y Video
- Tablas
- Formularios

### Unidad III: Conceptos Básicos de CSS (Sesiones 5-8)

- Introducción a CSS
- Color
- Texto
- Cajas (Modelo de caja)
- Listas, tablas y formularios
- Layouts (Flexbox, Grid)
- Imágenes
- Diseño responsivo (Media queries)

### Unidad IV: Conceptos Básicos de JavaScript (Sesiones 9-12)

- Introducción a JavaScript
- Variables y asignaciones
- Tipos de variables
- Los operadores
- Las condiciones
- Iteraciones
- Depuración de un programa
- Las cookies
- Interacción entre JavaScript, HTML y CSS
- Manipulación del DOM
- Lista de tareas (proyecto integrador)

### Unidad V: Primeros Pasos con Bootstrap (Sesiones 13-14)

- Introducción a Bootstrap
- Grid de Bootstrap
- Elementos de Bootstrap
- Componentes
- Formularios
- Flexbox
- Clases de utilidad

### Unidad VI: Publicación de un Sitio Web (Sesiones 15-18)

- Proyecto de creación de una página web completa
- Publicación en servidor (FTP)
- GitHub Pages
- Control de versiones con Git

## 🧠 Capacidades a Desarrollar

### Procedimentales

- Comprende los fundamentos del desarrollo web, conociendo la historia y evolución de la web
- Identifica los componentes fundamentales de una página web
- Entiende la importancia de los estándares web y la accesibilidad
- Conoce la sintaxis y semántica del lenguaje HTML
- Utiliza de manera adecuada las etiquetas básicas y avanzadas de HTML
- Diferencia entre etiquetas estructurales, de formato y multimedia
- Comprende el uso de selectores, propiedades y valores en CSS
- Implementa estilos responsivos con Flexbox, Grid y Media Queries
- Programa en JavaScript para agregar interactividad
- Manipula el DOM para cambiar contenido y estilos
- Construye diseños web responsivos con Bootstrap
- Integra componentes de Bootstrap (botones, menús, modales, carruseles)
- Publica páginas web en servidores y GitHub Pages
- Utiliza sistemas de control de versiones (Git)

### Actitudinales

- Desarrolla una actitud proactiva hacia el aprendizaje continuo
- Se mantiene actualizado sobre las últimas tendencias y herramientas
- Participa en comunidades de desarrollo web
- Fomenta la creatividad y la innovación en el diseño web
- Valora el diseño web accesible e inclusivo
- Adopta una actitud ética y responsable en la publicación de contenido
- Respeta los derechos de autor y las licencias de los recursos
- Promueve la colaboración y el trabajo en equipo
- Ofrece retroalimentación constructiva

## 📌 Normas del Repositorio

Para mantener la calidad y consistencia del contenido, este repositorio sigue las convenciones definidas en [AGENTS.md](./AGENTS.md).

### Principios fundamentales

1. **Accesibilidad**: Todo contenido debe ser accesible para personas con diferentes capacidades
2. **Semántica HTML**: Uso correcto de etiquetas semánticas (header, main, section, article, footer)
3. **Responsive Design**: El contenido debe adaptarse a diferentes tamaños de pantalla
4. **Código limpio**: Separación de estructura (HTML), presentación (CSS) y comportamiento (JavaScript)
5. **Comentarios en español**: El código debe estar comentado en español para facilitar el aprendizaje
