Ejercicio 2: Maquetación con Bootstrap Grid

Sesión 15 - Proyecto Final I

Prof. Ing. Juan M. Ferreira

Enunciado

Maqueta la página principal del proyecto utilizando HTML semántico y Bootstrap Grid. La página debe ser completamente responsiva.

  1. Estructura HTML semántica: <header>, <nav>, <main>, <section>, <footer>.
  2. Incluir el CDN de Bootstrap 5 en el <head>.
  3. Usar .container o .container-fluid para el contenido.
  4. Implementar un layout responsivo con filas (.row) y columnas (.col-*-*).
  5. Incluir al menos:
    • Un banner/hero principal con imagen y texto
    • Una sección de tarjetas con .card de Bootstrap
    • Un footer con información de contacto
  6. Aplicar utilidades de espaciado (m-*, p-*) y colores de fondo.
  7. Verificar que se vea correctamente en móvil, tablet y escritorio.

Requisitos

  • No usar CSS personalizado, solo clases de Bootstrap
  • Usar img-fluid para imágenes responsivas
  • Hacer commit y push de los cambios a GitHub
  • El diseño debe seguir el wireframe aprobado
