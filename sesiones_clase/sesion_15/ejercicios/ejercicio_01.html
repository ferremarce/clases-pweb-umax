Ejercicio 1: Wireframe y Git Init

Sesión 15 - Proyecto Final I

Prof. Ing. Juan M. Ferreira

Enunciado

Planifica la estructura de tu proyecto final siguiendo estos pasos:

  1. Dibuja un wireframe de la página principal del proyecto (puede ser en papel o digital).
  2. Define la estructura de carpetas del proyecto:
    • index.html - página principal
    • pages/ - páginas secundarias
    • css/ - archivos de estilo
    • js/ - scripts de JavaScript
    • images/ - recursos multimedia
  3. Inicializa un repositorio Git en la carpeta del proyecto.
  4. Crea el archivo .gitignore para excluir archivos innecesarios.
  5. Realiza el primer commit con la estructura base del proyecto.
  6. Crea un repositorio en GitHub y conecta el repositorio local.
  7. Sube el proyecto a GitHub con git push.

Requisitos

  • Tener Git instalado y configurado (user.name, user.email)
  • Contar con una cuenta en GitHub
  • Trabajar en equipo (3-5 integrantes)
  • Presentar el wireframe al docente para aprobación

Comandos Útiles

git init
git add .
git commit -m "feat: estructura inicial del proyecto"
git branch -M main
git remote add origin https://github.com/usuario/mi-proyecto.git
git push -u origin main
