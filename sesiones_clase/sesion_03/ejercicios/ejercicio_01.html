Ejercicio 1: Página con Navegación Interna

Sesión 3 - HTML II: Enlaces, Imágenes y Multimedia

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea una página web de una sola página (SPA) con navegación interna usando enlaces de anclas:

  • Una barra de navegación fija en la parte superior
  • Al menos 4 secciones con ids únicos: Inicio, About, Servicios, Contacto
  • Enlaces que navegen a cada sección usando #
  • Al menos una imagen en alguna sección

✅ Requisitos

  • Navbar con <nav> y enlaces <a>
  • Cada sección debe tener un id único
  • Los enlaces deben usar el formato href="#id"
  • Al menos una imagen con atributos src y alt
  • Estructura semántica (header, main, section, footer)

💡 Pistas

Ver pistas
  • El ancla se crea con el atributo id en el elemento destino
  • Usa href="#contacto" para ir a la sección con id="contacto"
  • La navbar puede usar position:fixed en CSS (lo verás más adelante)

🎯 Resultado Esperado

Una página con menú de navegación que al hacer clic en cada enlace baja suavemente a la sección correspondiente.
