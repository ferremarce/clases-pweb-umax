Ejercicio 3: Ampliar la Página Personal

Sesión 3 - HTML II: Enlaces, Imágenes y Multimedia

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Amplía tu página personal (creada en S2) añadiendo:

  • Barra de navegación con enlaces internos a secciones
  • Galería de imágenes
  • Video incrustado
  • Una página adicional "Acerca de" (acerca.html)

Este ejercicio conecta todas las sesiones: S2 (HTML base) → S3 (multimedia) → S4 (formularios) → S5 (CSS)

✅ Requisitos

  • 2 archivos HTML: index.html y acerca.html
  • Barra de navegación (nav) con enlaces a secciones (#inicio, #galeria, #video)
  • Mínimo 4 imágenes en una galería
  • Video embebido (YouTube o local)
  • Página "Acerca de" con información del autor
  • Navbar consistente en ambas páginas
  • Links funcionales entre index.html y acerca.html

💡 Pistas

Ver pistas
  • Enlaces internos: <a href="#galeria">Galería</a>
  • ID en secciones: <section id="galeria">
  • Para YouTube: compartir → insertar → copiar iframe
  • Imágenes de prueba: picsum.photos/300/200
  • Usa la misma estructura navbar en ambas páginas

🎯 Resultado Esperado

Tu página personal ahora tiene:

  • Menú de navegación que va a secciones internas
  • Galería visual de imágenes
  • Video relacionado contigo o tus intereses
  • Página "Acerca de" con información personal
  • Navegación fluida entre secciones

Este proyecto base se continuará estilizando en S5 con CSS.
