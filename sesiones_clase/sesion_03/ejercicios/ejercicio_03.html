Ejercicio 3: Sitio Multipágina con Multimedia

Sesión 3 - HTML II: Enlaces, Imágenes y Multimedia

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea un sitio web de 3 páginas interconectadas:

  • index.html: Página principal con navbar y presentación
  • videos.html: Página con video embebido de YouTube
  • contacto.html: Página con información de contacto
  • Todas las páginas deben tener navbar consistente

✅ Requisitos

  • 3 archivos HTML en la misma carpeta
  • Barra de navegación en las 3 páginas
  • Navbar debe tener enlaces a las 3 páginas
  • Videos.html debe incluir un iframe de YouTube
  • Al menos 2 imágenes en index.html
  • Links funcionales entre las páginas

💡 Pistas

Ver pistas
  • Usa el mismo navbar en las 3 páginas
  • Para YouTube: compartir → insertar → copiar iframe
  • El enlace de cada página es simplemente "videos.html"
  • Imágenes de prueba: picsum.photos/600/400

🎯 Resultado Esperado

Un sitio funcional donde el usuario puede navegar entre 3 páginas usando el menú. La página de videos muestra un video de YouTube embebido.
