Ejercicio 2: Galería de Imágenes

Sesión 3 - HTML II: Enlaces, Imágenes y Multimedia

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea una página con una galería de imágenes que muestre al menos 6 imágenes:

  • Usa la etiqueta <figure> y <figcaption> para cada imagen
  • Cada imagen debe tener atributo alt descriptivo
  • Las imágenes deben tener enlaces (al hacer clic ir a una URL externa)
  • Incluye un título principal y una descripción

✅ Requisitos

  • Mínimo 6 imágenes con figure/figcaption
  • Cada imagen debe tener atributo alt único y descriptivo
  • Cada imagen debe estar envuelta en un enlace <a>
  • Usa rutas relativas para las imágenes (pueden ser de placeholder)
  • Usa estructura semántica

💡 Pistas

Ver pistas
  • Usa servicios como picsum.photos para imágenes de prueba
  • Estructura: <a><figure><img><figcaption></figure></a>
  • picsum.photos/id/1/400/300 da una imagen de 400x300

🎯 Resultado Esperado

Una galería tipo grid con 6+ imágenes, cada una con su caption y enlace. Al hacer clic en una imagen debe abrir una URL (puede ser la misma imagen).
