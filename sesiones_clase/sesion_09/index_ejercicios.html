Sesión 9: JavaScript I - Variables, Tipos y Operadores

Ejercicios de Práctica

Prof. Ing. Juan M. Ferreira

Instrucciones Generales

En esta sesión practicarás los fundamentos de JavaScript: declaración de variables, tipos de datos primitivos y operadores aritméticos y de comparación.

Duración: 2.5 horas

Nota: Todos los ejercicios se resuelven usando prompt() para recibir datos y alert() o console.log() para mostrar resultados.
