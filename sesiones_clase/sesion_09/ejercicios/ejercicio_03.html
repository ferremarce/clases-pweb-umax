Ejercicio 3: Calculadora con Validación

Sesión 9 - JavaScript I

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea una calculadora robusta con validación de entrada y múltiples operaciones hasta que el usuario decida salir. El programa debe:

  1. Mostrar un menú de opciones al usuario:
    • 1 = Sumar
    • 2 = Restar
    • 3 = Multiplicar
    • 4 = Dividir
    • 5 = Salir
  2. Según la opción, pedir los números necesarios
  3. Validar que los ingresos sean números válidos
  4. Si es división, validar que el divisor no sea cero
  5. After showing result, ask if they want to continue

✅ Requisitos

  • Usar isNaN() para validar que el input sea un número
  • Usar Number() para convertir valores
  • Bucle que continúe hasta que el usuario elija 5
  • Mensaje claro de error para entradas inválidas
  • Mostrar mensaje de despedida al salir
  • Usar comparación estricta (===)

💡 Pistas

Ver pistas
  • Usa un bucle while que continúe mientras no sea 5
  • Para validar número: isNaN(Number(valor)) o valor === ""
  • Para validar división: verificar divisor antes de dividir
  • Usa prompt anidados o consecutivos según la operación

🎯 Resultado Ejemplo

Ejemplo de ejecución:

// Menú: "1) Sumar, 2) Restar, 3) Multiplicar, 4) Dividir, 5) Salir" // Ingresamos: 1
// Prompt: "Primer número:" // Ingresamos: 10
// Prompt: "Segundo número:" // Ingresamos: 5
// Alert: "Resultado: 15"
// Prompt: "¿Continuar? (s/n)" // Ingresamos: s
// ... (repite hasta que pongamos n o 5)
