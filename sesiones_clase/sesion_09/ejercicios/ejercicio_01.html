Ejercicio 1: Calculadora Básica

Sesión 9 - JavaScript I

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea un programa en JavaScript que funcione como una calculadora básica. El programa debe:

  1. Pedir al usuario que ingrese el primer número usando prompt()
  2. Pedir al usuario que ingrese el segundo número usando prompt()
  3. Realizar las cuatro operaciones aritméticas básicas:
    • Suma (+)
    • Resta (-)
    • Multiplicación (*)
    • División (/)
  4. Mostrar los cuatro resultados usando alert()

✅ Requisitos

  • 使用 prompt() para solicitar los números
  • Usar Number() para convertir strings a números
  • Realizar las 4 operaciones
  • Mostrar resultados con alert()
  • Manejar división por cero (mostrar mensaje "No se puede dividir por cero")

💡 Pistas

Ver pistas
  • Usa Number(valor) para convertir el string que retorna prompt a número
  • Para la división por cero, verifica si el divisor es 0 antes de dividir
  • Puedes usar concatenación de strings para construir el mensaje: "Texto " + variable

🎯 Resultado Esperado

Al ejecutar el programa, debería aparecer algo similar a:

// Primer prompt: "Ingresa el primer número:" // Ingresamos: 10

// Segundo prompt: "Ingresa el segundo número:" // Ingresamos: 5

// Alert con resultados: // Suma: 15 // Resta: 5 // Multiplicación: 50 // División: 2
