Ejercicio 2: Conversor de Temperatura

Sesión 9 - JavaScript I

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea un programa conversor de temperaturas que permita al usuario elegir la dirección de conversión. El programa debe:

  1. Preguntar al usuario qué quiere hacer:
    • 1 = Celsius a Fahrenheit
    • 2 = Fahrenheit a Celsius
  2. Si elige 1: pedir temperatura en Celsius y convertir a Fahrenheit
  3. Si elige 2: pedir temperatura en Fahrenheit y convertir a Celsius
  4. Mostrar el resultado con alert()

✅ Requisitos

  • Usar prompt() para la opción y la temperatura
  • Usar Number() para convertir valores
  • Fórmulas de conversión:
    • °F = (°C × 9/5) + 32
    • °C = (°F - 32) × 5/9
  • Mostrar resultado con 2 decimales usando toFixed(2)
  • Manejar opción inválida con mensaje de error

💡 Pistas

Ver pistas
  • Usa estructura if/else para decidir qué conversión hacer
  • Para mostrar 2 decimales: resultado.toFixed(2)
  • Comparación: opcion === "1" (usar string comparisons)

🎯 Resultado Esperado

Al ejecutar el programa:

// Prompt: "¿Qué conversión? (1) C→F, (2) F→C" // Ingresamos: 1

// Prompt: "Temperatura en Celsius:" // Ingresamos: 25

// Alert: // 25°C = 77.00°F
