Ejercicio 2: Gestor de Tareas

Sesión 9 - JavaScript I

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea un programa que funcione como un gestor simple de tareas. El programa debe solicitar los datos de una tarea y mostrar un análisis completo usando operadores aritméticos, de comparación y lógicos.

  1. Solicitar el nombre de la tarea usando prompt()
  2. Solicitar las horas estimadas para completarla (número)
  3. Solicitar la prioridad del 1 al 10 (número)
  4. Solicitar el número de subtareas (número)
  5. Calcular y mostrar:
    • Horas totales (estimadas + 10% de margen)
    • Horas por subtarea (si hay subtareas)
    • Si es tarea de alta prioridad (prioridad > 7)
    • Si es tarea larga (horas > 4)
    • Si es tarea crítica (alta prioridad Y larga)
  6. Mostrar un resumen completo con alert()

✅ Requisitos

  • Usar prompt() para solicitar los datos
  • Usar Number() para convertir strings a números
  • Usar operadores aritméticos (+, *, /) para los cálculos
  • Usar operadores de comparación (>, >=) para evaluar condiciones
  • Usar operador lógico AND (&&) para determinar si es crítica
  • Mostrar resultados con 1 decimal usando toFixed(1)
  • Usar alert() para mostrar el resumen
