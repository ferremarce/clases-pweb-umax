Ejercicio 2: Repartidor de Cuenta

Sesión 9 - JavaScript I

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea un programa que calcule cuánto debe pagar cada persona al dividir una cuenta de restaurante, incluyendo la propina. El programa debe:

  1. Solicitar el total de la cuenta usando prompt()
  2. Solicitar el porcentaje de propina (ej: 10, 15, 20)
  3. Solicitar el número de personas para dividir la cuenta
  4. Calcular:
    • Monto de la propina
    • Total con propina incluida
    • Total por persona
  5. Mostrar los resultados con alert()

✅ Requisitos

  • Usar prompt() para solicitar los datos
  • Usar Number() para convertir strings a números
  • Usar operadores aritméticos (+, *, /)
  • Mostrar resultados con 2 decimales usando toFixed(2)
  • Usar alert() para mostrar un resumen completo

💡 Pistas

Ver pistas
  • Propina = totalCuenta * (porcentajePropina / 100)
  • Total con propina = totalCuenta + propina
  • Por persona = totalConPropina / numeroPersonas
  • Para mostrar 2 decimales: resultado.toFixed(2)
