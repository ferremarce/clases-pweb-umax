Unidad IV - Fundamentos de JavaScript

¿Qué es JavaScript?

  • Lenguaje de programación interpretado de alto nivel
  • Multiplataforma: funciona en navegador, servidores, móviles
  • Lenguaje dinámico: no necesita declarar tipos
  • Esencial para la web: interactividad en páginas
⚡ Dato: JS es uno de los lenguajes más populares del mundo. Junto con HTML y CSS forma la base de la web.

¿Dónde se Ejecuta JavaScript?

🌐 Navegador

  • Chrome (V8)
  • Firefox (SpiderMonkey)
  • Edge (Chakra)
  • Safari (JavaScriptCore)

🖥️ Servidor

  • Node.js
  • Deno
  • Bun

Insertar JavaScript en HTML

1. Script inline (no recomendado)

<button onclick="alert('Hola!')">Click</button>

2. Script interno

<script>
    console.log("Hola mundo");
</script>

3. Script externo (recomendado)

<script src="js/script.js"></script>

Variables en JavaScript

Las variables son contenedores para almacenar datos.


Palabra Ámbito Reasignable Ejemplo
var Función ✅ Sí var nombre = "Ana";
let Bloque ✅ Sí let edad = 25;
const Bloque ❌ No const PI = 3.14;
💡 Recomendación: Usar const por defecto y let solo cuando necesites reasignar. Evitar var.

Ejemplo: Declarar Variables

// Usando var (evitar)
var nombre = "Juan";

// Usando let (para valores que cambian)
let edad = 30;
edad = 31; // ✅ Se puede reasignar

// Usando const (para valores fijos)
const PI = 3.14159;
// PI = 3.15; // ❌ Error: no se puede reasignar

// Constante con objeto (el contenido puede cambiar)
const usuario = { nombre: "Ana", edad: 25 };
usuario.edad = 26; // ✅ Esto sí funciona
// usuario = {}; // ❌ Error: no se puede reasignar

Tipos de Datos: null y undefined

null vs undefined

Tipo Significado Ejemplo
undefined Variable declarada pero sin valor let x;
null Valor "vacío" asignado intencionalmente let x = null; 

let nombre;        // undefined - no tiene valor
let apellido = null; // null - intencionalmente vacío

Operadores Aritméticos

Operador Nombre Ejemplo Resultado
+ Suma 5 + 3 8
- Resta 10 - 4 6
* Multiplicación 4 * 3 12
/ División 15 / 3 5
% Módulo (resto) 17 % 5 2
** Exponenciación 2 ** 3 8

Operadores de Asignación

Operador Ejemplo Equivalente a
= x = 5 x = 5
+= x += 3 x = x + 3
-= x -= 2 x = x - 2
*= x *= 4 x = x * 4
/= x /= 2 x = x / 2

Operadores de Comparación

Operador Nombre Ejemplo Resultado
== Igual (valor) 5 == "5" true
=== Estrictamente igual 5 === "5" false
!= Diferente (valor) 5 != "5" false
!== Estrictamente diferente 5 !== "5" true
> Mayor que 10 > 5 true
< Menor que 3 < 1 false
⚠️ Importante: Siempre usar === y !== para evitar errores de coerción de tipos.

Interactuar con el Usuario

alert() - Mostrar mensaje

alert("¡Hola, bienvenido!");

prompt() - Pedir datos al usuario

let nombre = prompt("¿Cuál es tu nombre?");
console.log("El nombre es: " + nombre);

console.log() - Mensaje en consola

console.log("Este mensaje aparece en la consola del navegador");
🎯 Pruébalo: Abre la consola (F12) y escribe: alert("¡Hola!")

Conversión de Tipos (Coerción)

Conversión Implícita (automática)

// String + Number = String
"5" + 3    // "53"

// Number + String = Number
5 + "3"    // 8

// Comparaciones con strings
"5" == 5   // true (convierte a número)
"5" === 5  // false (tipos diferentes)

Conversión Explícita (manual)

// Convertir a número
Number("5")      // 5
parseInt("5")    // 5
parseFloat("5.5")// 5.5

// Convertir a string
String(5)        // "5"
(5).toString()  // "5"

Boolean(1)       // true
Boolean(0)       // false

Ejemplo Completo

// Programa: Calculadora simple
let num1 = prompt("Ingresa el primer número:");
let num2 = prompt("Ingresa el segundo número:");

// Convertir a número
num1 = Number(num1);
num2 = Number(num2);

// Realizar operaciones
let suma = num1 + num2;
let resta = num1 - num2;
let producto = num1 * num2;
let division = num1 / num2;

// Mostrar resultados
alert("Resultados:\n" +
      "Suma: " + suma + "\n" +
      "Resta: " + resta + "\n" +
      "Producto: " + producto + "\n" +
      "División: " + division);
