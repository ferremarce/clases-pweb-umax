¿Qué es JavaScript?

JavaScript (JS) es el lenguaje de programación de la web. Permite crear páginas interactivas y dinámicas.


  • Creado por Brendan Eich en 1995
  • Se ejecuta en el navegador del usuario (Chrome V8, Firefox SpiderMonkey)
  • No confundir con Java (son lenguajes independientes)
  • Estándar actual: ECMAScript 6+ (ES6+)
💡 Clave: HTML define la estructura, CSS la presentación, JS el comportamiento.

Variables en JavaScript

Las variables son contenedores para almacenar datos.


Palabra Ámbito Reasignable Ejemplo
var Función ✅ Sí var nombre = "Ana";
let Bloque ✅ Sí let edad = 25;
const Bloque ❌ No const PI = 3.14;
💡 Recomendación: Usar const por defecto y let solo cuando necesites reasignar. Evitar var.

Ejemplo: Declarar Variables

// Usando var (evitar)
var nombre = "Juan";

// Usando let (para valores que cambian)
let edad = 30;
edad = 31; // ✅ Se puede reasignar

// Usando const (para valores fijos)
const PI = 3.14159;
// PI = 3.15; // ❌ Error: no se puede reasignar

// Constante con objeto (el contenido puede cambiar)
const usuario = { nombre: "Ana", edad: 25 };
usuario.edad = 26; // ✅ Esto sí funciona
// usuario = {}; // ❌ Error: no se puede reasignar

/= x /= 2 x = x / 2

Operadores de Comparación

Los operadores de comparación comparan dos valores y devuelven un booleano (true o false).

Operador Nombre Ejemplo Resultado
== Igualdad (débil) "5" == 5 true
=== Igualdad estricta "5" === 5 false
!= Desigualdad (débil) "5" != 5 false
!== Desigualdad estricta "5" !== 5 true
> Mayor que 10 > 5 true
< Menor que 10 < 5 false
>= Mayor o igual 5 >= 5 true
<= Menor o igual 5 <= 3 false
💡 Regla de oro: Usar siempre === y !== (comparación estricta) para evitar comportamientos inesperados por coerción automática.

Operadores Lógicos

Los operadores lógicos se usan para combinar condiciones y trabajar con valores booleanos.


AND (&&) — Todas las condiciones deben cumplirse

ExpresiónResultado
true && truetrue
true && falsefalse
false && truefalse
false && falsefalse

OR (||) — Al menos una condición debe cumplirse

ExpresiónResultado
true || truetrue
true || falsetrue
false || truetrue
false || falsefalse

NOT (!) — Invierte el valor booleano

let activo = true;
console.log(!activo);  // false

let esMayor = (10 > 5);
console.log(!esMayor); // false

// Doble negación (convierte a booleano)
console.log(!!"texto"); // true

Interactuar con el Usuario

alert() - Mostrar mensaje

alert("¡Hola, bienvenido!");

prompt() - Pedir datos al usuario

let nombre = prompt("¿Cuál es tu nombre?");
console.log("El nombre es: " + nombre);

console.log() - Mensaje en consola

console.log("Mensaje normal");
console.warn("Advertencia");
console.error("Error");
console.info("Información");
🎯 Pruébalo: Abre la consola (F12) y escribe: alert("¡Hola!")

Conversión de Tipos (Coerción)

Conversión Implícita (automática)

// String + Number = String
"5" + 3    // "53"

// Number + String = Number
5 + "3"    // 8

// Comparaciones con strings
"5" == 5   // true (convierte a número)
"5" === 5  // false (tipos diferentes)

Conversión Explícita (manual)

// Convertir a número
Number("5")      // 5
parseInt("5")    // 5
parseFloat("5.5")// 5.5

// Convertir a string
String(5)        // "5"
(5).toString()   // "5"

Boolean(1)       // true
Boolean(0)       // false

Comentarios en JavaScript

Los comentarios son texto ignorado por el motor de JS. Sirven para explicar el código y depurar.


Comentario de una línea

// Esto es un comentario de una línea
let nombre = "Ana"; // También al final de una línea

Comentario de varias líneas

/*
  Esto es un comentario
  de varias líneas.
  Útil para explicar bloques grandes.
*/
let precio = 100;
💡 Buen hábito: Comenta el por qué del código, no el qué (el código ya dice qué hace).

Buenas Prácticas y Nomenclatura

  • camelCase para variables: nombreUsuario, edadPersona
  • Nombres descriptivos: precioTotal en vez de pt
  • const por defecto, let solo si necesitas reasignar
  • El punto y coma (;) es opcional, pero sé consistente
  • Comenta el código para explicar el por qué, no el qué

Ejemplo Completo

// Programa: Calculadora simple
let num1 = prompt("Ingresa el primer número:");
let num2 = prompt("Ingresa el segundo número:");

// Convertir a número
num1 = Number(num1);
num2 = Number(num2);

// Realizar operaciones
let suma = num1 + num2;
let resta = num1 - num2;
let producto = num1 * num2;
let division = num1 / num2;

// Mostrar resultados
alert("Resultados:
" +
      "Suma: " + suma + "
" +
      "Resta: " + resta + "
" +
      "Producto: " + producto + "
" +
      "División: " + division);
