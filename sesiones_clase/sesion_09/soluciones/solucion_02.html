Solución: Ejercicio 2 - Repartidor de Cuenta

Sesión 9 - JavaScript I

Prof. Ing. Juan M. Ferreira

Código de la Solución

// Ejercicio 2: Repartidor de Cuenta // Solicitar datos al usuario let totalCuenta = Number(prompt("Total de la cuenta:")); let porcentajePropina = Number(prompt("Porcentaje de propina:")); let numPersonas = Number(prompt("Número de personas:")); // Calcular propina y totales let propina = totalCuenta * (porcentajePropina / 100); let totalConPropina = totalCuenta + propina; let porPersona = totalConPropina / numPersonas; // Mostrar resultados alert("Resumen de la cuenta:\n" + "Cuenta: $" + totalCuenta.toFixed(2) + "\n" + "Propina (" + porcentajePropina + "%): $" + propina.toFixed(2) + "\n" + "Total: $" + totalConPropina.toFixed(2) + "\n" + "Por persona: $" + porPersona.toFixed(2));
💡 Nota: Esta es UNA posible solución. Se usa toFixed(2) para mostrar solo 2 decimales. Observa cómo se usa concatenación con \n para crear saltos de línea.
