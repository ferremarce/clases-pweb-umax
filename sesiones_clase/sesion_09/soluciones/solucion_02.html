Código de la Solución

// Ejercicio 2: Conversor de Temperatura // Pedir opción de conversión let opcion = prompt("¿Qué conversión deseas?\n" + "1 = Celsius a Fahrenheit\n" + "2 = Fahrenheit a Celsius"); // Variables para temperatura y resultado let temperatura, resultado; // Procesar según opción if (opcion === "1") { // Celsius a Fahrenheit temperatura = Number(prompt("Temperatura en Celsius:")); resultado = (temperatura * 9/5) + 32; alert(temperatura + "°C = " + resultado.toFixed(2) + "°F"); } else if (opcion === "2") { // Fahrenheit a Celsius temperatura = Number(prompt("Temperatura en Fahrenheit:")); resultado = (temperatura - 32) * 5/9; alert(temperatura + "°F = " + resultado.toFixed(2) + "°C"); } else { // Opción inválida alert("Opción inválida. Debes elegir 1 o 2."); }
💡 Nota: Esta es UNA posible solución. Se usa toFixed(2) para mostrar solo 2 decimales. Nota que comparamos opcion con strings ("1", "2") porque prompt() retorna strings.
