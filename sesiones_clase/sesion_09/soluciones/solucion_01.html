Código de la Solución

// Ejercicio 1: Calculadora Básica // Solicitar dos números al usuario let num1 = prompt("Ingresa el primer número:"); let num2 = prompt("Ingresa el segundo número:"); // Convertir strings a números num1 = Number(num1); num2 = Number(num2); // Realizar operaciones let suma = num1 + num2; let resta = num1 - num2; let multiplicacion = num1 * num2; // División (verificar que no sea cero) let division; if (num2 === 0) { division = "No se puede dividir por cero"; } else { division = num1 / num2; } // MostrarResultados alert("Resultados:\n" + "Suma: " + suma + "\n" + "Resta: " + resta + "\n" + "Multiplicación: " + multiplicacion + "\n" + "División: " + division);
💡 Nota: Esta es UNA posible solución. Existen múltiples formas de implementar este ejercicio. Por ejemplo, podría usar una función para mostrar los resultados o validar los inputs usando isNaN().
