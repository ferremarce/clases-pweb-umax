Código Completo

HTML

<div class="layout">
    <header>
        <div class="logo">Mi Sitio</div>
        <nav>...</nav>
    </header>
    <aside>...</aside>
    <main>...</main>
    <footer>...</footer>
</div>

CSS - Grid

.layout {
    display: grid;
    grid-template-columns: 200px 1fr;
    grid-template-rows: auto 1fr auto;
    grid-template-areas:
        "header header"
        "sidebar main"
        "footer footer";
}

header { grid-area: header; }
aside   { grid-area: sidebar; }
main    { grid-area: main; }
footer { grid-area: footer; }

CSS - Flexbox (header)

header {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

CSS - Responsivo

@media (max-width: 768px) {
    .layout {
        grid-template-columns: 1fr;
        grid-template-areas:
            "header"
            "main"
            "footer";
    }
    aside { display: none; }
}

Explicación

  • Grid para el layout general de la página
  • grid-template-areas define la estructura visual
  • Flexbox para componentes (header, footer)
  • Media queries adapts a móvil
