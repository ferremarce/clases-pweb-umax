Solución: Ejercicio 1 - Navbar con Flexbox

Mi Sitio Web
  • Inicio
  • Productos
  • Servicios
  • Contacto

Código Completo

HTML

<nav class="navbar">
    <div class="logo">Mi Sitio Web</div>
    <ul class="menu">
        <li><a href="#">Inicio</a></li>
        <li><a href="#">Productos</a></li>
        <li><a href="#">Contacto</a></li>
    </ul>
</nav>

CSS

.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
    background: linear-gradient(135deg, #8B0000, #DC143C);
    padding: 15px 30px;
}

.logo {
    color: white;
    font-size: 1.5rem;
    font-weight: bold;
}

.menu {
    display: flex;
    gap: 20px;
    list-style: none;
}

.menu a {
    color: white;
    text-decoration: none;
    padding: 8px 16px;
    border-radius: 5px;
}

.menu a:hover {
    background: rgba(255,255,255,0.2);
}

Explicación

  • display: flex convierte el contenedor en flex container
  • justify-content: space-between separa el logo (izquierda) del menú (derecha)
  • align-items: center centra verticalmente
  • gap: 20px espacio entre elementos del menú
