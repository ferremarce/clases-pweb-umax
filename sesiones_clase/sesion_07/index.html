¿Por qué Flexbox y Grid?

Los métodos tradicionales (float, position) tienen limitaciones:

Métodos Antiguos

  • Float: Requiere clearfix, comportamiento impredecible
  • Position: Absoluto sale del flujo, difícil de mantener
  • Table: No es semántico, mala accesibilidad

Flexbox + Grid

  • Flexbox: Layout unidimensional (fila o columna)
  • Grid: Layout bidimensional (filas y columnas)
  • Juntos: Solución completa para cualquier diseño

Flexbox trabaja con dos ejes:

Main Axis

Cross Axis
  • Main Axis: Eje principal (horizontal por defecto)
  • Cross Axis: Eje perpendicular (vertical por defecto)

Propiedades de Items

Propiedades que puedes aplicar a cada elemento hijo:

order

order: 1;  /* Por defecto es 0 */
1 (order:3)
2 (order:1)
3 (order:2)

flex-grow

flex-grow: 1;  /* Crece para llenar espacio */

flex-shrink

flex-shrink: 0;  /* No se encoge */

Atajo: flex

Propiedad combinada:

Sintaxis

flex: flex-grow flex-shrink flex-basis;

Ejemplos

flex: 1 0 auto;    /* grow, shrink, basis */
flex: 1 1 0;     /* Equivale a flex: 1 */
flex: 0 0 auto;     /* Equivale a fixed */

Introducción a CSS Grid

Grid permite crear layouts bidimensionales:

Contenedor Grid

.grid {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    gap: 20px;
}

Unidades de Grid

Unidades específicas para grids:

fr (fracción)

grid-template-columns: 1fr 2fr 1fr;
/* 25% 50% 25% */

auto

grid-template-columns: auto 1fr auto;
/* Ajusta al contenido, resto */

repeat()

grid-template-columns: repeat(3, 1fr);
/* Equivale a 1fr 1fr 1fr */

gap

Espacio entre celdas:

gap: 20px;
row-gap: 10px;
column-gap: 30px;

grid-template-areas

Define áreas con nombres:

CSS

.layout {
    display: grid;
    grid-template-areas: "header header" "sidebar main" "footer footer";
    grid-template-columns: 200px 1fr;
    grid-template-rows: auto 1fr auto;
}

HTML

<div style="grid-area: header">Header</div>
<div style="grid-area: sidebar">Sidebar</div>
<div style="grid-area: main">Main</div>

Comparativa: Flexbox vs Grid

Flexbox

Una dimensión

  • fila O columna
  • Navbars
  • Items de tarjetas
  • Centrado

Grid

Dos dimensiones

  • filas Y columnas
  • Galería
  • Layouts complejos
  • Páginas completas
💡 Consejo: Usa ambos juntos. Grid para el layout general, Flexbox para componentes.

Diseño Responsivo

Grid facilita el diseño responsivo:

Grid Responsivo

.galeria {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 20px;
}
  • auto-fit: Ajusta el número de columnas
  • minmax(250px, 1fr): Mínimo 250px, máximo fracción
  • Sin media queries: Se adapta automáticamente

Ejemplo Práctico: Layout Completo

Estructura

.contenedor {
    display: grid;
    grid-template-columns: 250px 1fr;
    grid-template-rows: auto 1fr auto;
    grid-template-areas: "header header" "sidebar main" "footer footer";
    min-height: 100vh;
}

header { grid-area: header; }
aside  { grid-area: sidebar; }
main   { grid-area: main; }
footer { grid-area: footer; }

Buenas Prácticas

✅ Hacer:

  • Usar Flexbox para componentes unidimensionales
  • Usar Grid para layouts de página
  • Combinar ambos en el mismo proyecto
  • Usar unidades fr y auto-fit para responsivo
  • Usar gap para espaciado

❌ Evitar:

  • Float para layouts principales
  • position: absolute para layout general
  • Usar solo Flexbox para grids complejos
  • Olvidar gap (usar margin)
