Ejercicio 2: Galería con CSS Grid

Enunciado

Crea una galería de imágenes responsiva usando CSS Grid.

Requisitos:

  • Contenedor con display: grid
  • grid-template-columns con repeat()
  • gap para espaciado
  • Diseño responsivo con media queries (opcional)

HTML Sugerido

<div class="galeria">
    <div class="item">Imagen 1</div>
    <div class="item">Imagen 2</div>
    <div class="item">Imagen 3</div>
    <div class="item">Imagen 4</div>
    <div class="item">Imagen 5</div>
    <div class="item">Imagen 6</div>
</div>

CSS Sugerido

.galeria {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 15px;
}

.item {
    background: #DC143C;
    color: white;
    padding: 40px;
    text-align: center;
    border-radius: 8px;
}

CSS Avanzado: Responsivo

.galeria {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    gap: 15px;
}

auto-fit ajusta automáticamente las columnas según el espacio disponible.
