Ejercicio 1: Navbar con Flexbox

Enunciado

Crea una barra de navegación usando Flexbox conLogo, menú y botón de acción.

Requisitos:

  • Contenedor con display: flex
  • justify-content: space-between para separar logo y menú
  • align-items: center para centrar verticalmente
  • Menú con display: flex y gap

HTML Sugerido

<nav class="navbar">
    <div class="logo">Mi Sitio</div>
    <ul class="menu">
        <li><a href="#">Inicio</a></li>
        <li><a href="#">Productos</a></li>
        <li><a href="#">Contacto</a></li>
    </ul>
</nav>

CSS Sugerido

.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
    background: linear-gradient(135deg, #8B0000, #DC143C);
    padding: 15px 25px;
}

.menu {
    display: flex;
    gap: 20px;
    list-style: none;
}

Resultado Esperado

Mi Sitio
  • Inicio
  • Productos
  • Contacto
