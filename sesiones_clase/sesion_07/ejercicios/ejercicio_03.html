Ejercicio 3: Layout Completo

Enunciado

Combina Flexbox y Grid para crear un layout completo de página web.

Requisitos:

  • Grid para el layout principal (header, sidebar, main, footer)
  • grid-template-areas para nombrar secciones
  • Flexbox para el menú de navegación
  • Footer con elementos alineados con Flexbox

HTML

<div class="layout">
    <header>...</header>
    <aside>...</aside>
    <main>...</main>
    <footer>...</footer>
</div>

CSS - Grid

.layout {
    display: grid;
    grid-template-columns: 200px 1fr;
    grid-template-rows: auto 1fr auto;
    grid-template-areas:
        "header header"
        "sidebar main"
        "footer footer";
}

header { grid-area: header; }
aside   { grid-area: sidebar; }
main    { grid-area: main; }
footer { grid-area: footer; }

CSS - Flexbox para componentes

/* Menú del header */
header {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

/* Footer */
footer {
    display: flex;
    justify-content: center;
    gap: 20px;
}
