Ejercicio 1: Interactividad JS y GitHub Pages

Sesión 16 - Proyecto Final II

Prof. Ing. Juan M. Ferreira

Enunciado

Agrega interactividad a tu proyecto final e implementa al menos 2 de las siguientes funcionalidades usando JavaScript:

  1. Menú interactivo: resaltar la página activa en el navbar usando JavaScript.
  2. Modal de confirmación: usar Bootstrap Modal con JS para confirmar acciones (eliminar, enviar, etc.).
  3. localStorage: guardar y recuperar preferencias del usuario (tema oscuro, nombre, etc.).
  4. Validación de formulario: validar campos con JavaScript antes de enviar.
  5. Filtro de contenido: buscar y filtrar elementos dinámicamente.

Luego, publica el proyecto en GitHub Pages siguiendo estos pasos:

  • Ir a Settings → Pages en el repositorio de GitHub
  • Seleccionar "Deploy from branch" y elegir main
  • Guardar y esperar 1-2 minutos
  • Visitar la URL generada para verificar

Requisitos

  • El código JavaScript debe estar en un archivo script.js externo
  • Usar addEventListener para los eventos
  • Todos los integrantes del grupo deben contribuir con commits
  • Verificar que el sitio publicado funcione correctamente
