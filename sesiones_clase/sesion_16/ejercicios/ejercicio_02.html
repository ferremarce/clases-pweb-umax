Ejercicio 2: Proyecto Final Completo

Sesión 16 - Proyecto Final II

Prof. Ing. Juan M. Ferreira

Enunciado

Completa el proyecto final asegurando que cumpla con todos los requisitos:

  1. HTML semantico: todas las paginas usan etiquetas semanticas (header, nav, main, section, footer).
  2. Bootstrap responsivo: el layout se adapta a movil, tablet y escritorio usando Grid.
  3. Componentes Bootstrap: usar al menos Navbar, Cards y un Modal o formulario.
  4. JavaScript interactivo: al menos 2 funcionalidades JS funcionando correctamente.
  5. localStorage: persistir al menos un tipo de datos (preferencias, contactos, etc.).
  6. Git: commits de todos los integrantes con mensajes descriptivos.
  7. GitHub Pages: sitio publicado y funcionando.
  8. Presentacion: preparar una demo de 5-7 minutos explicando el proyecto.

Criterios de Evaluacion

  • HTML semantico y accesible (20%)
  • CSS + Bootstrap responsivo (20%)
  • JavaScript e interactividad (20%)
  • Uso de Git y GitHub (15%)
  • Publicacion en GitHub Pages (10%)
  • Presentacion y trabajo en equipo (15%)

Entrega

  • Link a GitHub Pages funcionando
  • Repositorio Git con commits de todos los integrantes
  • Presentacion en clase (5-7 min)
