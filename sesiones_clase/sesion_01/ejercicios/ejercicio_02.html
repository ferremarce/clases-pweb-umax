Ejercicio 2: Explorando Etiquetas de Texto

Sesión 1 - Fundamentos de la Web

📝 Enunciado

Crear una página que demuestre el uso de diferentes etiquetas HTML para formatear texto. Esta página servirá como referencia rápida de etiquetas.

✅ Requisitos

Crear una página que muestre ejemplos de TODAS estas etiquetas:

  • Encabezados: <h1> hasta <h6>
  • Párrafos: <p>
  • Salto de línea: <br>
  • Línea horizontal: <hr>
  • Texto importante: <strong>
  • Texto enfatizado: <em>

Además, organiza el contenido en secciones claras con títulos descriptivos.

💡 Pistas

Ver pistas
  • Pista 1: Cada tipo de etiqueta debe estar claramente identificada con un <h2> o <h3>
  • Pista 2: Después de cada ejemplo, incluye una breve explicación de para qué sirve
  • Pista 3: Usa <hr> para separar las diferentes secciones
  • Pista 4: Experimenta: cambia el texto y observa cómo cambia el resultado

🎯 Estructura Sugerida

Sección Contenido
Encabezados Mostrar h1 hasta h6
Párrafos 2-3 párrafos de ejemplo
Formato de texto strong, em, br, hr
