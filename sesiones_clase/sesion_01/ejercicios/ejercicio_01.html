Ejercicio 1: Mi Primera Página Personal

Sesión 1 - Fundamentos de la Web

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crear una página web personal básica que incluya información sobre ti. La página debe tener una estructura HTML correcta y usar las etiquetas fundamentales de texto.

✅ Requisitos

  • Estructura HTML5 completa con <!DOCTYPE html>
  • Atributo lang="es" en la etiqueta <html>
  • Meta charset UTF-8 y meta viewport
  • Título en la pestaña del navegador usando <title>
  • Encabezado principal <h1> con tu nombre
  • Al menos 2 párrafos <p> con información personal
  • Un subtítulo <h2> para una sección
  • Al menos 2 usos de <strong> para texto importante
  • Al menos 2 usos de <em> para texto enfatizado
  • Una línea horizontal <hr> para separar secciones

💡 Pistas

Ver pistas
  • Pista 1: Empieza siempre con la estructura básica: <!DOCTYPE html>, <html>, <head>, <body>
  • Pista 2: El contenido va dentro de <body>, los metadatos van en <head>
  • Pista 3: Solo debe haber un <h1> por página (para el título principal)
  • Pista 4: Usa <strong> para palabras clave y <em> para énfasis o títulos de obras

🎯 Resultado Esperado

Tu página debería verse similar a esto:

+----------------------------------+
|  ¡Bienvenidos!                   |
|  Juan Pérez                      |
|                                  |
|  Hola, soy estudiante de...      |
|  Me gusta aprender sobre...      |
|  -------------------------------- |
|                                  |
|  Mis Gustos                      |
|  Me encanta la música y el...    |
+----------------------------------+

🚀 Código Base (opcional)

Puedes usar este código como punto de partida:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Página Personal</title>
</head>
<body>
    <!-- Tu contenido aquí -->
</body>
</html>
