Ejercicio 1: Mi Página de Presentación

Sesión 2 - HTML I: Estructura, Texto y Listas

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea una página web de presentación personal que incluya:

  • Estructura semántica completa (header, main, footer)
  • Un encabezado principal (h1) con tu nombre
  • Una sección "Sobre mí" con un párrafo descriptivo
  • Aplicar estilos inline CSS: color al título y padding al párrafo
  • Un pie de página con el año y tu nombre

✅ Requisitos

  • Debe tener la estructura DOCTYPE HTML5 correcta
  • Debe incluir meta charset UTF-8 y viewport
  • Solo un elemento h1 por página
  • El contenido debe ser real y personalizado
  • Debe usar etiquetas semánticas (header, main, footer)
  • Usar al menos style="color: ..." y style="padding: ..." como inline CSS

💡 Pistas

Ver pistas
  • Usa la plantilla básica de HTML5 vista en clase
  • header va antes de main, y footer después
  • El párrafo va dentro de una etiqueta <p>

🎯 Resultado Esperado

Una página simple con tu nombre como título, una descripción breve sobre ti, y un pie de página con copyright.
