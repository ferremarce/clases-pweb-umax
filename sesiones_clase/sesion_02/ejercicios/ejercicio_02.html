Ejercicio 2: Lista de Compras con Categorías

Sesión 2 - HTML I: Estructura, Texto y Listas

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea una lista de compras organizada por categorías usando listas anidadas:

  • Una lista desordenada principal con categorías
  • Cada categoría debe tener sublistas con los artículos
  • Usa listas ordenadas para las cantidades
  • Incluye al menos 3 categorías con 3+ artículos cada una

✅ Requisitos

  • Mínimo 3 categorías principales (ul)
  • Cada categoría debe tener sublista (ul u ol)
  • Al menos una sublista debe ser ordenada
  • La estructura debe ser clara y legible
  • Usar indentación correcta

💡 Pistas

Ver pistas
  • Estructura: <ul> → <li> → <ul>
  • Usa type="A" para listas ordenadas con letras
  • Para cantidades, usa <ol type="1">

🎯 Resultado Esperado

Una página que muestre:

  • Frutas: Manzana, Plátano, Naranja
  • Lácteos: Leche (2), Queso (1), Yogur (3)
  • Panadería: Pan, Bollos, Croissants
