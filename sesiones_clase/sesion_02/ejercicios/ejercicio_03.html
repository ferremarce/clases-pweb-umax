Ejercicio 3: Página de Recetas

Sesión 2 - HTML I: Estructura, Texto y Listas

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea una página web completa de recetas con estructura semántica profesional:

  • Header con título del sitio
  • Una barra de navegación (nav) con enlaces
  • Multiple secciones (section) con artículos de recetas
  • Encabezados jerárquicos (h1, h2, h3)
  • Listas para ingredientes y pasos
  • Estilos inline CSS: aplicar color, border y padding a recetas
  • Footer con información de contacto

✅ Requisitos

  • Solo un h1 (título principal del sitio)
  • Mínimo 2 secciones con artículos de recetas
  • Cada receta debe tener: título, descripción, ingredientes (lista), pasos (lista)
  • Navbar con al menos 3 enlaces
  • Footer con información realista
  • Código indentado correctamente
  • Usar <details> y <summary> para información adicional de cada receta
  • Usar <abbr> para definir términos como "kcal", "g", "ml"
  • Aplicar inline CSS con style="color: ...", style="border: ..." y style="padding: ..."

💡 Pistas

Ver pistas
  • Estructura: body → header → nav → main → section → article
  • Usa <ul> para ingredientes y <ol> para pasos
  • Losnav links pueden ser "#receta1", "#receta2", etc.

🎯 Resultado Esperado

Una página similar a un blog de cocina con múltiples recetas, cada una con su lista de ingredientes y pasos numerados.
