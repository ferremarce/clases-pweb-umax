Ejercicio 1: Formulario de Contacto Estilizado

Sesión 8 - CSS IV

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea un formulario de contacto con estilos CSS modernos y professionnels. El formulario debe incluir:

  1. Campo de nombre (text)
  2. Campo de correo electrónico (email)
  3. Campo de teléfono (tel)
  4. Campo de mensaje (textarea)
  5. Botón de enviar

✅ Requisitos

  • Input de texto: borde, padding, border-radius
  • Estado focus: cambio de color de borde y box-shadow
  • Estado hover en botones
  • Botón con estilo primario (fondo rojo UMAX)
  • Cursor pointer en botones
  • Transiciones suaves (transition 0.3s)
  • Uso de box-sizing: border-box

💡 Pistas

Ver pistas
  • Usa selectors: input:focus, button:hover
  • Para el color primario: #8B0000
  • padding recomendado: 10px 15px
  • border-radius: 4px o 6px

🎯 Resultado Esperado

El formulario debe verse profesional con:

  • Inputs con bordes redondeados
  • Al hacer click en un input, el borde cambia a rojo
  • El botón cambia de color al pasar el mouse
  • Transiciones suaves al interactuar
