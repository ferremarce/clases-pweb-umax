Ejercicio 2: Tabla de Datos Responsiva

Sesión 8 - CSS IV

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea una tabla de datos HTML con estilos CSS atractivos y diseño responsivo. La tabla debe incluir:

  1. Encabezados de columna: Nombre, Cargo, Departamento, Salario
  2. Al menos 5 filas de datos
  3. Diseño visual mejorado para escritorio y móvil

✅ Requisitos

  • border-collapse: collapse en table
  • Zebra striping: tr:nth-child(even)
  • Efecto hover en filas
  • Bordes en celdas
  • Para móvil: overflow-x: auto en contenedor
  • padding en celdas
  • Encabezado con fondo oscuro

💡 Pistas

Ver pistas
  • Zebra: background-color en filas pares
  • Contenedor con overflow-x: auto habilita scroll horizontal
  • header con background: #8B0000
  • white-space: nowrap para texto largo

🎯 Resultado Esperado

  • Filas交替颜色 (zebra striping)
  • Al pasar el mouse, la fila cambia de color
  • En móvil, la tabla tiene scroll horizontal
  • Encabezado visible con fondo oscuro
