Instrucciones Generales

En esta sesión practicarás la estilización avanzada de formularios, tablas y el diseño responsivo usando media queries, Flexbox y Grid.

Duración: 2.5 horas

Nota: Usa el archivo HTML proporcionado y crea hojas de estilo CSS externas. Incluye siempre el meta viewport tag.
