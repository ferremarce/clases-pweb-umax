Estilización de Formularios

Los formularios requieren estilos específicos para mejorar la experiencia del usuario.


input, textarea, select {
    width: 100%;
    padding: 10px;
    margin: 8px 0;
    border: 2px solid #ccc;
    border-radius: 4px;
    font-size: 1rem;
    box-sizing: border-box;
}

Estados de Formulario

Los inputs tienen estados que pueden estilizarse:


/* Input normal */
input { border: 1px solid #ccc; }

/* Input con foco */
input:focus {
    border-color: #DC143C;
    outline: none;
    box-shadow: 0 0 5px rgba(220,20,60,0.3);
}

/* Input inválido */
input:invalid {
    border-color: red;
}

/* Input válido */
input:valid {
    border-color: green;
}

Botones Personalizados

.btn {
    padding: 12px 24px;
    border: none;
    border-radius: 6px;
    font-size: 1rem;
    cursor: pointer;
    transition: background 0.3s;
}

.btn-primary {
    background: #8B0000;
    color: white;
}

.btn-primary:hover {
    background: #DC143C;
}

Estilización de Tablas

table {
    width: 100%;
    border-collapse: collapse;
}

th, td {
    padding: 12px;
    text-align: left;
    border-bottom: 1px solid #ddd;
}

/* Zebra striping */
tr:nth-child(even) {
    background-color: #f2f2f2;
}

/* Hover */
tr:hover {
    background-color: #ddd;
}

¿Qué es Diseño Responsivo?

El diseño responsivo permite que un sitio web se adapte a diferentes tamaños de pantalla.


  • 📱 Móvil: phones (< 576px)
  • 📲 Tablet: tablets (576px - 992px)
  • 💻 Escritorio: desktop (> 992px)
💡 Clave: Un solo código HTML/CSS para todos los dispositivos.

Meta Viewport Tag

Etiqueta esencial para que funcione el diseño responsivo:


<meta name="viewport" 
      content="width=device-width, initial-scale=1.0">

  • width=device-width: ancho igual al dispositivo
  • initial-scale=1.0: zoom inicial 100%

Media Queries

Las media queries aplican estilos según características del dispositivo:


/* Estilos para móvil (por defecto) */
body { font-size: 14px; }

/* Estilos para tablet */
@media (min-width: 576px) {
    body { font-size: 16px; }
}

/* Estilos para escritorio */
@media (min-width: 992px) {
    body { font-size: 18px; }
}

Breakpoints Comunes

Dispositivo Ancho Breakpoint
Móvil < 576px default
Tablet ≥ 576px min-width: 576px
Escritorio ≥ 992px min-width: 992px
Grande ≥ 1200px min-width: 1200px

Mobile-First Design

El enfoque mobile-first consiste en:


  1. Diseñar primero para móvil
  2. Usar min-width para añadir características
  3. Progressive enhancement

/* Mobile-first: estilos base */
.container { width: 100%; }

/* Tablet */
@media (min-width: 576px) {
    .container { width: 540px; }
}

/* Desktop */
@media (min-width: 992px) {
    .container { width: 960px; }
}

Flexbox Responsivo

.menu {
    display: flex;
    flex-direction: column;
}

/* En tablet/desktop: fila */
@media (min-width: 768px) {
    .menu {
        flex-direction: row;
        justify-content: space-between;
    }
}

  • flex-direction: column: apilar vertical
  • flex-direction: row: alinear horizontal

Grid Responsivo

.grid {
    display: grid;
    grid-template-columns: 1fr; /* 1 columna móvil */
    gap: 20px;
}

@media (min-width: 576px) {
    .grid {
        grid-template-columns: repeat(2, 1fr); /* 2 columnas */
    }
}

@media (min-width: 992px) {
    .grid {
        grid-template-columns: repeat(3, 1fr); /* 3 columnas */
    }
}

Imágenes Flexibles

Las imágenes deben adaptarse al contenedor:


img {
    max-width: 100%;
    height: auto;
}

  • max-width: 100%: nunca exceder el contenedor
  • height: auto: mantener proporción

Ejemplo Completo Responsivo

<div class="layout">
    <header class="header">...</header>
    <nav class="nav">...</nav>
    <main class="content">...</main>
    <aside class="sidebar">...</aside>
    <footer class="footer">...</footer>
</div>

/* Mobile: todo apilado */
.layout {
    display: flex;
    flex-direction: column;
}

/* Desktop: grid */
@media (min-width: 992px) {
    .layout {
        display: grid;
        grid-template-columns: 200px 1fr 250px;
    }
}
