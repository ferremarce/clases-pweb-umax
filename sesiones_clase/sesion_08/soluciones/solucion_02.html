Código de la Solución

/* Contenedor con scroll horizontal */ .tabla-contenedor { overflow-x: auto; } /* Estilos base de tabla */ table { width: 100%; border-collapse: collapse; min-width: 600px; } /* Encabezado */ th { background: #8B0000; color: white; padding: 12px; text-align: left; } /* Celdas */ td { padding: 12px; border-bottom: 1px solid #ddd; } /* Zebra striping - filas pares */ tr:nth-child(even) { background-color: #f2f2f2; } /* Hover en filas */ tr:hover { background-color: #ddd; }
💡 Nota: El contenedor con overflow-x: auto permite que en dispositivos móviles la tabla tenga scroll horizontal sin romper el layout. min-width asegura que las columnas no se compriman demasiado.
