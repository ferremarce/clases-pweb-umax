Código de la Solución

/* Variables CSS */ :root { --color-primario: #8B0000; --color-secundario: #DC143C; --bg-claro: #f5f5f5; } /* Reset básico */ * { box-sizing: border-box; margin: 0; padding: 0; } /* Container */ .container { max-width: 1200px; margin: 0 auto; padding: 0 20px; } /* Header */ header { background: var(--color-primario); color: white; padding: 20px 0; } /* Navegación - Mobile first */ nav ul { list-style: none; display: flex; flex-direction: column; gap: 10px; } /* Navegación - Desktop */ @media (min-width: 768px) { nav ul { flex-direction: row; justify-content: flex-end; } } /* Galería - Grid responsivo */ .galeria { display: grid; grid-template-columns: 1fr; gap: 20px; } /* Galería - Tablet */ @media (min-width: 576px) { .galeria { grid-template-columns: repeat(2, 1fr); } } /* Galería - Desktop */ @media (min-width: 992px) { .galeria { grid-template-columns: repeat(3, 1fr); } } /* Tarjeta de proyecto */ .proyecto { background: white; border-radius: 8px; box-shadow: 0 2px 8px rgba(0,0,0,0.1); overflow: hidden; } /* Imagen */ .proyecto img { width: 100%; height: auto; object-fit: cover; } /* Footer */ footer { background: #333; color: white; text-align: center; padding: 20px; margin-top: 40px; }
💡 Nota: El diseño usa mobile-first: los estilos base son para móvil, y los media queries agregan características para tablet (min-width: 576px) y desktop (min-width: 992px).
