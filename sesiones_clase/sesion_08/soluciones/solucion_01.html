Código de la Solución

/* CSS para formulario */ .formulario { max-width: 500px; margin: 0 auto; padding: 20px; } /* Estilos base para inputs */ .formulario input, .formulario textarea { width: 100%; padding: 12px 15px; margin: 8px 0 16px; border: 2px solid #ccc; border-radius: 6px; font-size: 1rem; box-sizing: border-box; transition: border-color 0.3s, box-shadow 0.3s; } /* Estado focus */ .formulario input:focus, .formulario textarea:focus { border-color: #DC143C; outline: none; box-shadow: 0 0 8px rgba(220,20,60,0.3); } /* Botón */ .btn/enviar { background: #8B0000; color: white; padding: 12px 24px; border: none; border-radius: 6px; font-size: 1rem; cursor: pointer; transition: background 0.3s; } /* Hover del botón */ .btn/enviar:hover { background: #DC143C; }
💡 Nota: Los colores institucionales de UMAX son #8B0000 (primario) y #DC143C (secundario). El diseño usa transiciones suaves para una mejor experiencia de usuario.
