¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo que define la presentación visual de documentos HTML.

¿Para qué sirve?

  • Colores: Definir paletas de color para el sitio
  • Tipografía: Elegir fuentes, tamaños y estilos
  • Layout: Crear disposición de elementos
  • Animaciones: Añadir movimiento y transiciones
  • Responsive: Adaptar a diferentes dispositivos
💡 Dato clave: HTML estructura el contenido, CSS define la apariencia. Juntos crean páginas web completas.

Sintaxis de CSS

Una regla CSS tiene tres partes principales:

Anatomía de una regla CSS

selector {
    propiedad: valor;
    propiedad: valor;
}

Ejemplo concreto:

h1 {
    color: #DC143C;
    font-size: 2rem;
    text-align: center;
}
  • Selector: Indica qué elemento estilizar
  • Propiedad: Característica a modificar
  • Valor: Configuración específica

Formas de Aplicar CSS

Existen tres métodos para incluir estilos en una página:

1. CSS Inline (en línea)

Se aplica directamente en la etiqueta HTML:

<p style="color: blue; font-size: 18px;">Texto</p>

⚠️ No recomendado: difícil de mantener

2. CSS Interno (en el head)

Se define dentro de la página HTML:

<style>
    p { color: blue; font-size: 18px; }
</style>

3. CSS Externo (archivo separado)

Se vincula un archivo .css externo:

<link rel="stylesheet" href="css/estilos.css">

✅ Mejor práctica: mantenible y reutilizable

Selectores Básicos

Los selectores indican qué elementos HTML serán estilizados:

Selector de Etiqueta

Aplica a todas las etiquetas especificadas:

p {
    color: #333;
}

Selector de Clase (.)

Aplica a elementos con esa clase:

.tarjeta {
    background: white;
    padding: 20px;
}
<p class="tarjeta">Contenido</p>

Selector de ID (#)

Aplica a un elemento único:

#encabezado-principal {
    font-size: 2rem;
}
<h1 id="encabezado-principal">Título</h1>

Selectores Combinadores

Los combinadores permiten seleccionar elementos según su relación:

Descendiente (espacio)

Selecciona elementos dentro de otro:

article p {
    color: #333;
}

Hijo directo (>)

Selecciona hijos directos:

ul > li {
    list-style: none;
}

Hermano adyacente (+)

Selecciona el elemento que sigue:

h2 + p {
    font-style: italic;
}

Colores en CSS

CSS ofrece varias formas de especificar colores:

Nombre

color: red;

Más simple pero limitado

Hexadecimal

color: #FF5733;

Formato más usado

RGB

color: rgb(255, 87, 51);

Mezcla de rojo, verde, azul

RGBA

color: rgba(255, 87, 51, 0.5);

RGB con transparencia

HSL (Hue, Saturation, Lightness)

color: hsl(0, 100%, 50%);

Tono, saturación y luminosidad. Más intuitivo para ajustar colores.

Paleta de Colores UMAX

Usaremos esta paleta institucional en el curso:

Colores UMAX

  • Primario: #8B0000 (Rojo oscuro)
  • Secundario: #DC143C (Rojo crimson)
  • Texto: #1a1a1a (Casi negro)
  • Fondo: #f5f7fb (Gris claro)

Variables CSS

Definimos variables para reutilizar colores:

:root {
    --color-primario: #8B0000;
    --color-secundario: #DC143C;
    --color-texto: #1a1a1a;
}

Propiedades de Texto

CSS ofrece múltiples propiedades para estilizar texto:

Familia de Fuente

font-family: "Segoe UI", sans-serif;

Tamaño de Fuente

font-size: 16px;
font-size: 1.5rem;
font-size: 150%;

Peso de Fuente

font-weight: normal;  /* 400 */
font-weight: bold;    /* 700 */
font-weight: 300;    /* light */

Alineación

text-align: left;
text-align: center;
text-align: justify;

Más Propiedades de Texto

Altura de Línea

line-height: 1.6;  /*Recomendado: 1.5-1.8 */

Decoración de Texto

text-decoration: underline;
text-decoration: line-through;
text-decoration: none;

Transformación

text-transform: uppercase;   /* MAYÚSCULAS */
text-transform: lowercase;   /* minúsculas */
text-transform: capitalize; /* Primera Mayúscula */

Espaciado

letter-spacing: 2px;
word-spacing: 5px;

Modelo de Caja (Box Model)

Todo elemento HTML es una caja compuesta por:

margin (exterior)
border
padding
content (contenido)
  • Content: El contenido (texto, imágenes)
  • Padding: Espacio entre contenido y borde
  • Border: Línea alrededor del padding
  • Margin: Espacio exterior hacia otros elementos

Propiedades del Modelo de Caja

Padding (espacio interno)

padding: 20px;              /* 4 lados */
padding: 10px 20px;        /* vertical horizontal */
padding-top: 10px;
padding-right: 20px;
padding-bottom: 10px;
padding-left: 20px;

Margin (espacio externo)

margin: 20px;
margin: 10px auto;        /* centrar horizontalmente */

Border (borde)

border: 2px solid #333;
border-radius: 8px;       /* bordes redondeados */

Unidades de Medida

CSS ofrece diferentes unidades para medidas:

Unidades Absolutas

  • px (píxeles): Valor fijo, no cambia
font-size: 16px;
border: 1px solid black;

Unidades Relativas

  • em: Relativo al font-size del padre
  • rem: Relativo al font-size del root (html)
  • %: Porcentaje respecto al padre
font-size: 1.5rem;
width: 80%;
margin: 1em;
💡 Consejo: Usa rem para tamaños de texto y % para layouts. Evita px para texto que deba ser accesible.

Ejemplo Práctico

Vamos a estilizar una tarjeta:

HTML

<article class="tarjeta">
    <h2>Título de la Tarjeta</h2>
    <p>Contenido de ejemplo.</p>
</article>

CSS

.tarjeta {
    background: white;
    padding: 20px;
    border: 1px solid #ddd;
    border-radius: 8px;
    margin: 20px;
    box-shadow: 0 2px 8px rgba(0,0,0,0.1);
}

.tarjeta h2 {
    color: #8B0000;
    font-size: 1.5rem;
    margin-bottom: 10px;
}

.tarjeta p {
    color: #333;
    line-height: 1.6;
}

Buenas Prácticas

✅ Hacer:

  • Usar archivo CSS externo
  • Usar clases en lugar de IDs para estilos
  • Usar unidades relativas (rem, %) para accesibilidad
  • Definir variables CSS para colores
  • Comentar el código CSS
  • Usar selectores específicos y no demasiado genéricos

❌ Evitar:

  • CSS inline en etiquetas
  • Usar !important excesivamente
  • Selectores demasiado específicos
  • Olvidar el box-sizing: border-box

Cascada y Especificidad

Cuando hay conflictos, CSS usa reglas para decidir qué estilo aplicar:

Orden de Cascada

  1. Origen (inline > interno > externo)
  2. Especificidad (ID > clase > etiqueta)
  3. Orden en el código (el último gana)

Ejemplo de Especificidad

/* Menos específico */
p { color: blue; }

/* Más específico */
article p.destacado { color: red; }

/* Gana porque tiene mayor especificidad */
