Solución: Ejercicio 3 - Estilos para Página Personal

Sesión 5 - CSS I: Fundamentos, Color y Texto

Prof. Ing. Juan M. Ferreira

Código de la Solución

Este ejemplo muestra cómo estilizar una página personal completa (basada en la estructura de S2-S4) con CSS externo.

HTML (index.html - página personal)

<!DOCTYPE html> <html lang="es"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Mi Página Personal</title> <link rel="stylesheet" href="estilos.css"> <link href="https://fonts.googleapis.com/css2?family=Poppins:wght@400;600;700&display=swap" rel="stylesheet"> </head> <body> <header class="encabezado"> <h1>Juan Pérez</h1> <nav class="navegacion"> <a href="#sobre-mi">Sobre Mí</a> <a href="#proyectos">Proyectos</a> <a href="#contacto">Contacto</a> </nav> </header> <main> <section id="sobre-mi" class="tarjeta"> <h2>Sobre Mí</h2> <p>Soy estudiante de Ingeniería en Sistemas...</p> </section> <section id="proyectos" class="tarjeta"> <h2>Mis Proyectos</h2> <p>Aquí destaco mis trabajos...</p> </section> </main> <footer class="pie"> <p>© 2024 Juan Pérez</p> </footer> </body> </html>

CSS (estilos.css)

/* Variables CSS - Colores UMAX */ :root { --color-primario: #8B0000; --color-secundario: #DC143C; --color-fondo: #f5f7fb; --color-texto: #1a1a1a; --color-texto-claro: #666; --sombra: 0 4px 12px rgba(0,0,0,0.15); --radio: 12px; } /* Estilos generales */ body { font-family: 'Poppins', sans-serif; background: var(--color-fondo); color: var(--color-texto); line-height: 1.8; margin: 0; padding: 0; } /* Encabezado */ .encabezado { background: linear-gradient(135deg, var(--color-primario), var(--color-secundario)); color: white; padding: 2rem; text-align: center; } .encabezado h1 { font-size: 2rem; margin-bottom: 1rem; } /* Navegación */ .navegacion { margin-top: 1rem; } .navegacion a { color: white; text-decoration: none; margin: 0 10px; font-weight: 600; } .navegacion a:hover { text-decoration: underline; } /* Main y secciones */ main { max-width: 800px; margin: 2rem auto; padding: 0 1rem; } /* Tarjetas */ .tarjeta { background: white; padding: 1.5rem; border-radius: var(--radio); box-shadow: var(--sombra); margin-bottom: 1.5rem; } .tarjeta h2 { color: var(--color-primario); font-size: 1.5rem; margin-bottom: 0.5rem; } /* Footer */ .pie { background: var(--color-texto); color: white; text-align: center; padding: 1rem; margin-top: 2rem; }
💡 Nota: Esta es UNA posible solución. Se usa la página personal creada en S2-S4 como base, y se le aplica CSS externo con variables, modelo de caja, colores UMAX y tipografía. Este ejercicio conecta todas las sesiones anteriores: HTML (contenido) + CSS (estilo) = página web completa.
