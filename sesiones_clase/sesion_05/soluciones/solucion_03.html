Este ejemplo muestra cómo estilizar la página personal creada en la Sesión 3 (Ejercicio 3) con CSS externo.

CSS (estilos.css)

/* Variables CSS - Colores UMAX */ :root { --color-primario: #8B0000; --color-secundario: #DC143C; --color-fondo: #f5f7fb; --color-texto: #1a1a1a; --color-texto-claro: #666; --color-fondo-tarjeta: #ffffff; --sombra: 0 4px 12px rgba(0,0,0,0.15); --radio: 12px; } /* Estilos generales */ * { box-sizing: border-box; } body { font-family: 'Poppins', sans-serif; background: var(--color-fondo); color: var(--color-texto); line-height: 1.8; margin: 0; padding: 0; } /* Encabezado con gradiente */ .encabezado { background: linear-gradient(135deg, var(--color-primario), var(--color-secundario)); color: white; padding: 2rem; text-align: center; } .encabezado h1 { font-size: 2.5rem; margin-bottom: 0.5rem; } /* Navegación */ .navegacion { margin-top: 1rem; } .navegacion a { color: white; text-decoration: none; margin: 0 15px; font-weight: 600; font-size: 1rem; padding: 0.5rem 1rem; border-radius: 20px; transition: background 0.3s;; } .navegacion a:hover { background: rgba(255,255,255,0.2); } /* Main y secciones */ main { max-width: 900px; margin: 2rem auto; padding: 0 1rem; } /* Tarjetas */ .tarjeta { background: var(--color-fondo-tarjeta); padding: 1.5rem; border-radius: var(--radio); box-shadow: var(--sombra); margin-bottom: 1.5rem; } .tarjeta h2 { color: var(--color-primario); font-size: 1.5rem; margin-bottom: 1rem; } .tarjeta h3 { color: var(--color-primario); font-size: 1.2rem; margin-top: 1rem; margin-bottom: 0.5rem; } .tarjeta ul { list-style: disc; padding-left: 20px; line-height: 1.8; } /* Galería */ .galeria { display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); gap: 1rem; } .galeria img { width: 100%; height: 150px; object-fit: cover; border-radius: 8px; transition: transform 0.3s;; } .galeria img:hover { transform: scale(1.05); } /* Contenedor de video */ .contenedor-video { position: relative; padding-bottom: 56.25%; height: 0; overflow: hidden; border-radius: var(--radio); } .contenedor-video iframe { position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: none; border-radius: var(--radio); } /* Footer */ .pie { background: var(--color-texto); color: white; text-align: center; padding: 1.5rem; margin-top: 2rem; }
💡 Nota: Esta es UNA posible solución. Se estiliza la página personal creada en S3 (Ejercicio 3) con CSS externo, usando las secciones existentes (nav, galería, video, footer) como estructura base. Este ejercicio conecta todas las sesiones anteriores: HTML (contenido de S2-S4) + CSS (estilo de S5) = página web completa.
