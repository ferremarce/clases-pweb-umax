CSS (estilos.css)

/* Selector de etiqueta */ h1 { color: #8B0000; text-align: center; font-family: Segoe UI, sans-serif; } /* Selector de clase */ .tarjeta { background: rgb(245, 247, 251); padding: 20px; margin: 20px auto; max-width: 500px; border-radius: 8px; box-shadow: 0 2px 8px rgba(0,0,0,0.1); } /* Selector de ID */ #mi-boton { background: hsl(350, 100%, 46%); color: white; padding: 12px 24px; border: none; border-radius: 4px; cursor: pointer; display: block; margin: 20px auto; } /* Selector combinador descendiente */ .lista-estilos li { padding: 8px; color: #333; } /* Selector combinador hijo directo */ .lista-estilos > li { border-bottom: 1px solid #ddd; }
💡 Nota: Esta es UNA posible solución. Se demuestra el uso de selectores de etiqueta, clase, ID y combinadores, así como colores en formato hexadecimal, rgb() y hsl().
