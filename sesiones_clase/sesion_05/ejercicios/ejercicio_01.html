Ejercicio 1: Selectores y Colores Básicos

Sesión 5 - CSS I: Fundamentos, Color y Texto

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea una página web simple que demuestre el uso de selectores CSS básicos y diferentes formatos de color. La página debe tener un título, un párrafo introductorio y una lista de elementos.

✅ Requisitos

  • Usar selector de etiqueta para <h1> con color en formato hexadecimal
  • Usar selector de clase para estilizar una tarjeta con color en formato rgb()
  • Usar selector de ID para un botón con color en formato hsl()
  • Crear una lista con estilos usando selectores combinadores
  • El archivo CSS debe ser externo (crear archivo separado)

💡 Pistas

Ver pistas
  • Crea un archivo estilos.css y vincúlalo con <link rel="stylesheet" href="estilos.css">
  • Para el h1 usa: color: #8B0000;
  • Para la clase usa: background: rgb(245, 247, 251);
  • Para el ID del botón usa: background: hsl(350, 100%, 46%);

🎯 Resultado Esperado

Una página con título en rojo oscuro, una tarjeta con fondo gris claro, un botón en rojo, y una lista con estilos personalizados.
