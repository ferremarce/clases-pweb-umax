Ejercicio 2: Estilos Tipográficos

Sesión 5 - CSS I: Fundamentos, Color y Texto

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea una sección de blog simple con varios artículos que demuestren el uso de propiedades tipográficas de CSS. La página debe tener un diseño limpio y legible.

✅ Requisitos

  • Aplicar font-family distinto para títulos y texto
  • Usar font-size con unidades rem para los encabezados
  • Aplicar font-weight para negritas y pesos
  • Usar text-align para centrar el título principal
  • Aplicar line-height para mejorar la lectura
  • Usar text-transform para convertir texto a mayúsculas
  • Archivo CSS externo llamado estilos.css

💡 Pistas

Ver pistas
  • Usa una fuente de Google Fonts como "Poppins" o "Open Sans"
  • Importa la fuente con @import o en el HTML
  • Para el line-height usa: line-height: 1.8;
  • Para text-transform: text-transform: uppercase;

🎯 Resultado Esperado

Una página tipo blog con títulos grandes en negrita, texto fluido con buen interlineado, y una apariencia profesional.
