Solución: Ejercicio 2 - Contador de Visitas

Sesión 12 - JavaScript IV

Prof. Ing. Juan M. Ferreira

Código de la Solución

<!-- JavaScript --> <script> // Obtener visitas actuales (o 0 si no existe) let visitas = parseInt(sessionStorage.getItem("visitas")) || 0; // Incrementar visitas++; // Guardar sessionStorage.setItem("visitas", visitas); // Mostrar en página document.getElementById("contador").textContent = visitas; </script> <!-- HTML --> <div class="contador-app"> <h2>Contador de Visitas</h2> <p>Visitas en esta sesión: <span id="contador">0</span></p> <button onclick="reiniciar()">Reiniciar</button> </div> // Función para reiniciar function reiniciar() { sessionStorage.removeItem("visitas"); location.reload(); }
💡 Nota: La diferencia clave es que sessionStorage se borra al cerrar la pestaña, mientras que localStorage persiste.
