Sesión 12: JavaScript IV - Manipulación del DOM

Ejercicios de Práctica

Prof. Ing. Juan M. Ferreira

Instrucciones Generales

En esta sesión practicarás la manipulación del DOM: seleccionar elementos, manejar eventos, modificar contenido, crear y eliminar elementos dinámicamente.

Duración: 2.5 horas

Nota: Todos los datos se manejan en memoria (arrays). No uses localStorage ni sessionStorage.
