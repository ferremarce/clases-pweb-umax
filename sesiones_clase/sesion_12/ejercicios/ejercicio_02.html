Ejercicio 2: Contador de Visitas con sessionStorage

Sesión 12 - JavaScript IV

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea un contador de visitas que use sessionStorage para contar las visitas en la sesión actual.

  1. Mostrar un contador de visitas
  2. Incrementar el contador cada vez que se cargue la página
  3. Mostrar "Visitas en esta sesión: X"
  4. El contador debe reiniciarse al cerrar la pestaña

✅ Requisitos

  • Usar sessionStorage (no localStorage)
  • sessionStorage.getItem() para obtener valor
  • sessionStorage.setItem() para guardar
  • Incrementar el contador en 1 cada visita
  • Manejar el caso de primera visita (null)

💡 Pistas

Ver pistas
  • let visitas = parseInt(sessionStorage.getItem("visitas")) || 0
  • visitas++
  • sessionStorage.setItem("visitas", visitas)
  • La diferencia con localStorage es que sessionStorage se borra al cerrar

🎯 Resultado Esperado

  • Primera visita: "Visitas en esta sesión: 1"
  • Recargar página: "Visitas en esta sesión: 2"
  • Cerrar pestaña y abrir nueva: "Visitas en esta sesión: 1"
  • Abrir otra pestaña nueva: "Visitas en esta sesión: 1"
