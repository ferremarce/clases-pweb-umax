Ejercicio 3: To-Do List Completa con Persistencia

Sesión 12 - JavaScript IV

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea una lista de tareas completa con todas las funcionalidades y persistencia en localStorage.

  1. Input para escribir tarea + botón agregar
  2. Lista que muestre las tareas
  3. Click para marcar completada/tachado
  4. Botón para eliminar cada tarea
  5. Todas las acciones deben persistir

✅ Requisitos

  • Usar localStorage con JSON
  • Estructura de datos: { texto, completada }
  • Función agregarTarea(texto)
  • Función marcarCompletada(index)
  • Función eliminarTarea(index)
  • Función guardarTareas()
  • Función cargarTareas()
  • Renderizar tareas al iniciar
  • Tecla Enter para agregar

💡 Pistas

Ver pistas
  • Array de objetos: [{ texto: "tarea1", completada: false }]
  • JSON.stringify() para guardar, JSON.parse() para recuperar
  • tareas.push() para agregar, tareas.splice() para eliminar
  • Renderizar en un loop: forEach o for
  • classList.toggle("completada") para tachar

🎯 Resultado Esperado

  • Agregar tarea -> aparece en lista (persiste)
  • Click tarea -> tachado (persiste)
  • Eliminar X ->.remove() (persiste)
  • Recargar -> todas las acciones persisten
