Ejercicio 1: Calculadora Visual con DOM

Sesión 12 - JavaScript IV

Prof. Ing. Juan M. Ferreira

Enunciado

Crea una calculadora con interfaz visual usando HTML, CSS y JavaScript. Reutiliza las funciones de la sesión anterior pero ahora conectadas al DOM.

  1. Dos campos <input type="number"> para los operandos
  2. Cuatro botones: Sumar, Restar, Multiplicar, Dividir
  3. Un <div> para mostrar el resultado
  4. Cada botón usa addEventListener para ejecutar la operación

Requisitos

  • Usar document.getElementById() para seleccionar elementos
  • Usar addEventListener() para los eventos de los botones
  • Mostrar el resultado con textContent en el DOM
  • Manejar división por cero mostrando mensaje de error
  • Las funciones de operación deben estar separadas

Pistas

Ver pistas
  • let num1 = Number(document.getElementById("num1").value);
  • document.getElementById("resultado").textContent = "Resultado: " + suma;
  • document.getElementById("btnSumar").addEventListener("click", () => { ... });
  • Reutiliza las funciones sumar(a, b), restar(a, b), etc. de la sesión 11

Resultado Esperado

  • Interfaz con dos inputs y cuatro botones
  • Al hacer clic en "Sumar" se muestra la suma en pantalla
  • Al hacer clic en "Dividir" con 0, muestra mensaje de error
  • El diseño es visualmente atractivo (usa el CSS de práctica)
