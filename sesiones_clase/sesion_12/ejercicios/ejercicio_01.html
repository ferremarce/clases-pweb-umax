Ejercicio 1: Lista de Tareas Simple con localStorage

Sesión 12 - JavaScript IV

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea una lista de tareas simple que guarde las tareas en localStorage para que persistan al recargar la página.

  1. Input para escribir una tarea
  2. Botón "Agregar" para añadir la tarea
  3. Lista que muestre las tareas agregadas
  4. Al recargar la página, las tareas deben seguir ahí

✅ Requisitos

  • Usar localStorage.setItem() para guardar
  • Usar localStorage.getItem() para recuperar
  • Guardar las tareas como array JSON
  • Al cargar la página, renderizar las tareas guardadas
  • Al agregar tarea, guardar en localStorage

💡 Pistas

Ver pistas
  • let tareas = JSON.parse(localStorage.getItem("tareas")) || []
  • localStorage.setItem("tareas", JSON.stringify(tareas))
  • Al iniciar, verificar si hay tareas guardadas
  • Crear una función renderizarTareas()

🎯 Resultado Esperado

  • Agregar tarea -> aparece en la lista
  • Recargar página -> las tareas siguen ahí
  • Cerrar navegador y volver -> las tareas siguen ahí
