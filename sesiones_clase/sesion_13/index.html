Bootstrap I

Grid System y Utilidades

Objetivos de la Sesión

Al finalizar esta sesión, serás capaz de:

  • Comprender qué es Bootstrap y cómo integrarlo mediante CDN
  • Utilizar el sistema de contenedores, filas y columnas
  • Aplicar el sistema de 12 columnas con clases responsivas
  • Usar breakpoints para diseños adaptables
  • Aplicar utilidades de espaciado, tipografía y colores
Repaso: En sesiones anteriores aprendimos CSS Flexbox, Grid y media queries para diseño responsivo.

Qué es Bootstrap

Bootstrap es el framework CSS más popular para desarrollar sitios responsivos.


  • Creado por Twitter (2011), ahora open source
  • Sistema de Grid de 12 columnas
  • Componentes listos para usar
  • Diseño responsivo por defecto
  • Compatibilidad cross-browser
Versión actual: Bootstrap 5 (no requiere jQuery).

Configuración con CDN

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Página</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <!-- Contenido -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
CDN: Content Delivery Network — el archivo se sirve desde servidores rápidos, no necesita descarga.

Contenedores

Clase Comportamiento Uso
.container Ancho fijo responsivo Diseño estándar
.container-fluid 100% del ancho Diseño completo 

<div class="container">
    <!-- Contenido centrado con márgenes -->
</div>

<div class="container-fluid">
    <!-- Contenido a ancho completo -->
</div>

Sistema de Grid

Bootstrap divide la página en 12 columnas.


  • .row — crea una fila
  • .col-* — columnas (1 a 12)
  • Las columnas deben sumar 12 por fila
  • Se adaptan automáticamente al contenedor

<div class="row">
    <div class="col-6">Mitad</div>
    <div class="col-6">Mitad</div>
</div>

Breakpoints Responsivos

Breakpoint Ancho mínimo Prefijo
Extra small <576px .col-
Small ≥576px .col-sm-
Medium ≥768px .col-md-
Large ≥992px .col-lg-
Extra large ≥1200px .col-xl-

Grid Responsivo

<!-- 3 columnas iguales en desktop -->
<div class="row">
    <div class="col-4">Columna 1</div>
    <div class="col-4">Columna 2</div>
    <div class="col-4">Columna 3</div>
</div>

<!-- 1 col móvil, 3 cols en tablet+ -->
<div class="row">
    <div class="col-12 col-md-4">...</div>
    <div class="col-12 col-md-4">...</div>
    <div class="col-12 col-md-4">...</div>
</div>

Offset (Espaciado entre Columnas)

Los offsets desplazan columnas a la derecha.


<!-- Centrar columna -->
<div class="row">
    <div class="col-md-6 offset-md-3">
        Contenido centrado
    </div>
</div>

<!-- Espacio entre columnas -->
<div class="row">
    <div class="col-md-4">...</div>
    <div class="col-md-4 offset-md-4">...</div>
</div>

Ordenamiento (Order)

Controla el orden visual de las columnas.


<div class="row">
    <div class="col">1 (primero en HTML)</div>
    <div class="col order-3">2 (tercero visual)</div>
    <div class="col order-1">3 (primero visual)</div>
</div>

<!-- order-first y order-last -->
<div class="col order-last">Último</div>
<div class="col order-first">Primero</div>

Utilidades de Espaciado

Clases para margin (m) y padding (p).


Fórmula Significado
m-3 margin: 1rem en todos los lados
mt-3 margin-top: 1rem
mx-auto margin horizontal automático
p-4 padding: 1.5rem
py-2 padding vertical: 0.5rem

Valores: 0, 1 (0.25rem), 2 (0.5rem), 3 (1rem), 4 (1.5rem), 5 (3rem)

Utilidades de Texto y Color

<!-- Alineación -->
text-start / text-center / text-end

<!-- Tamaños -->
fs-1 (grande) ... fs-6 (pequeño)

<!-- Peso -->
fw-bold / fw-normal / fw-light

<!-- Colores de texto -->
text-primary / text-success / text-danger
text-white / text-dark / text-muted

<!-- Colores de fondo -->
bg-primary / bg-success / bg-warning
bg-danger / bg-info / bg-light / bg-dark

<!-- Mayúsculas/minúsculas -->
text-uppercase / text-lowercase / text-capitalize
