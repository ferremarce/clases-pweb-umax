Bootstrap I

Fundamentos y Grid System

Unidad V - Framework CSS

¿Qué es Bootstrap?

  • Framework CSS de código abierto
  • Desarrollado por Twitter
  • Componentes responsive ya listos
  • Sistema de Grid de 12 columnas
  • Componentes: Navbar, Cards, Modales, etc.
🎯 Ventaja: Prototipado rápido y diseño consistente.

Ventajas de Bootstrap

✅ Ventajas

  • Diseño responsivo automático
  • Componentes pre-diseñados
  • Gran comunidad y documentación
  • Ahorra tiempo de desarrollo
  • Consistencia en el código

⚠️ Desventajas

  • Todos los sitios se ven similares
  • Peso del archivo CSS
  • Curva de aprendizaje
  • Personalización limitada

CDN vs Descarga Local

CDN (推荐)

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

Descarga Local

<link href="css/bootstrap.min.css" rel="stylesheet">
<script src="js/bootstrap.bundle.min.js"></script>
💡 CDN: Más rápido, no requiere descarga. Local: Funciona sin internet.

Configuración Básica

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Página</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <!-- Contenido -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

Sistema de Grid

Bootstrap usa un sistema de 12 columnas.


  • .row - Crea una fila
  • .col-* - Columnas (1-12)
  • Las columnas suman 12
  • Se adaptan automáticamente

<div class="row">
    <div class="col-6">Mitad</div>
    <div class="col-6">Mitad</div>
</div>

Breakpoints

Breakpoint Ancho mínimo Clase
Extra small - .col-
Small 576px .col-sm-
Medium 768px .col-md-
Large 992px .col-lg-
Extra large 1200px .col-xl-

Ejemplos de Grid

<!-- 3 columnas iguales -->
<div class="row">
    <div class="col-4">Columna 1</div>
    <div class="col-4">Columna 2</div>
    <div class="col-4">Columna 3</div>
</div>

<!-- Responsive: 1 col móvil, 3 cols desktop -->
<div class="row">
    <div class="col-12 col-md-4">...</div>
    <div class="col-12 col-md-4">...</div>
    <div class="col-12 col-md-4">...</div>
</div>

Offset (Espaciado)

Los offsets crean espacio entre columnas.


<!-- Offset de 4 columnas -->
<div class="row">
    <div class="col-md-4">...</div>
    <div class="col-md-4 offset-md-4">...</div>
</div>

<!-- Centrar columna -->
<div class="row">
    <div class="col-md-6 offset-md-3">Centrado</div>
</div>

Ordenamiento (Order)

Controla el orden visual de las columnas.


<div class="row">
    <div class="col">1</div>
    <div class="col order-3">2 (tercero)</div>
    <div class="col order-1">3 (primero)</div>
</div>

<!-- order-first y order-last -->
<div class="col order-last">Último</div>
<div class="col order-first">Primero</div>

Utilidades de Espaciado

Clases de utilidad para margin y padding.


Propiedad Dirección Valor
m (margin) t, b, l, r, x, y 0-5, auto
p (padding) t, b, l, r, x, y 0-5 

mt-3 = margin-top: 1rem
mb-2 = margin-bottom: 0.5rem
p-4 = padding: 1.5rem
mx-auto = margin-left:auto; margin-right:auto
