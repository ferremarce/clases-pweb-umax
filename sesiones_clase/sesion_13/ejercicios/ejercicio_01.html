Ejercicio 1: Layout Básico con Grid

📝 Enunciado

Crea un layout básico usando el sistema de Grid de Bootstrap. La página debe mostrar:

  1. Un header con título
  2. Una fila con 2 columnas iguales (6+6)
  3. Una fila con 3 columnas iguales (4+4+4)
  4. Una fila con 4 columnas iguales (3+3+3+3)

✅ Requisitos

  • Usar solo clases de Bootstrap (sin CSS personalizado)
  • Usar .container-centered
  • Usar .row y .col-*
  • Cada columna debe tener contenido visible
  • Diseño funcional en cualquier tamaño

💡 Pistas

Ver pistas
  • <div class="container"> para el contenedor
  • <div class="row"> para las filas
  • <div class="col-6"> para 2 columnas
  • <div class="col-4"> para 3 columnas
  • <div class="col-3"> para 4 columnas

🎯 Resultado Esperado

  • Fila 1: 2 columnas de igual ancho
  • Fila 2: 3 columnas de igual ancho
  • Fila 3: 4 columnas de igual ancho
