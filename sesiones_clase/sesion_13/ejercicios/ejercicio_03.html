Ejercicio 3: Dashboard con Grid

📝 Enunciado

Crea un dashboard con sidebar, contenido principal y footer usando solo clases de Bootstrap. El layout debe incluir:

  1. Navbar简单
  2. Sidebar (2 columnas) + Contenido principal (10 columnas)
  3. 4 tarjetas de estadísticas en el contenido
  4. Footer

✅ Requisitos

  • Usar .row con .col-2 y .col-10 para sidebar
  • Usar .col-3 para 4 tarjetas
  • Usar .card component de Bootstrap
  • Sidebar oculta en móvil (col-12 en móvil)
  • Footer fijo al final

💡 Pistas

Ver pistas
  • col-12 col-lg-2 = sidebar responsivo
  • col-12 col-lg-10 = contenido
  • <div class="card"> para tarjetas
  • card-title, card-body para contenido
  • mt-4, mb-4 para espaciado

🎯 Resultado Esperado

  • Móvil: sidebar arriba, contenido después
  • Desktop: sidebar izquierdo, contenido derecho
  • 4 tarjetas en fila (3+3+3+3)
