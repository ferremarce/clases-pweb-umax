Ejercicio 2: Página Responsiva con Grid

📝 Enunciado

Crea una página responsiva que se adapte correctamente a móvil, tablet y escritorio. La página debe incluir:

  1. Header con título
  2. Sección "Sobre nosotros" - 2 columnas (texto + imagen)
  3. Sección "Servicios" - 3 columnas
  4. Footer simple

✅ Requisitos

  • Usar .col-12 .col-md-6 para responsivo (móvil 1 col, desktop 2)
  • Usar .col-12 .col-md-4 para servicios (móvil 1 col, desktop 3)
  • Usar clases utilitarias de Bootstrap
  • Imágenes con .img-fluid
  • Footer con bg-dark y texto blanco

💡 Pistas

Ver pistas
  • col-12 col-md-6 = 100% móvil, 50% desktop
  • col-12 col-md-4 = 100% móvil, 33% desktop
  • img-fluid para imágenes responsivas
  • text-center para centrar texto
  • py-4 para padding vertical

🎯 Resultado Esperado

  • Móvil: todo apilado en 1 columna
  • Desktop: 2 columnas en "Sobre nosotros", 3 en "Servicios"
  • Imágenes se adaptan al contenedor
