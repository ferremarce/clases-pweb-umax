Ejercicio 1: Calculadora con Funciones

Sesión 11 - JavaScript III

Prof. Ing. Juan M. Ferreira

Enunciado

Crea una calculadora que use funciones para cada operación aritmética. El menú se controla con prompt() y los resultados se muestran con alert(). No uses HTML ni eventos del DOM.

  1. Función sumar(a, b) — devuelve a + b
  2. Función restar(a, b) — devuelve a - b
  3. Función multiplicar(a, b) — devuelve a * b
  4. Función dividir(a, b) — devuelve a / b (manejar división por cero)
  5. Función calcular(operacion) — pide dos números, ejecuta la operación recibida como callback y muestra el resultado
  6. Bucle while con menú para elegir operación o salir

Requisitos

  • Declarar al menos una operación como arrow function
  • Usar calcular(operacion) donde la operación se pasa como callback
  • Validar división por cero con retorno temprano
  • Menú numérico con prompt() y switch
  • Todo el código en un solo archivo HTML dentro de <script>

Pistas

Ver pistas
  • const sumar = (a, b) => a + b;
  • function calcular(callback) { ... callback(a, b) ... }
  • if (b === 0) return "Error: división por cero";
  • switch(opcion) { case "1": calcular(sumar); break; ... }

Resultado Esperado

  • Al cargar la página aparece el menú con prompt
  • Seleccionar operación, ingresar números, ver resultado en alert
  • Dividir por cero muestra mensaje de error
  • El menú se repite hasta elegir "Salir"
