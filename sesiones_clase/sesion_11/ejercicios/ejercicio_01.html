Ejercicio 1: Calculadora con Funciones

Sesión 11 - JavaScript III

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea una calculadora que use funciones para realizar las operaciones. Los botones deben responder a eventos click y mostrar el resultado en la página.

  1. Dos campos de entrada (input type="number") para los operandos
  2. 4 botones para las operaciones: Sumar, Restar, Multiplicar, Dividir
  3. Un área para mostrar el resultado
  4. Cada operación debe implementarse como una función

✅ Requisitos

  • Usar funciones con parámetros para cada operación
  • Usar addEventListener para los botones
  • Mostrar resultado en un elemento HTML (no alert)
  • Manejar división por cero
  • Diseño visual atractivo con CSS

💡 Pistas

Ver pistas
  • function sumar(a, b) { return a + b; }
  • document.getElementById("boton").addEventListener("click", funcion)
  • Number(document.getElementById("input").value)
  • resultadoDiv.textContent = "Resultado: " + resultado

🎯 Resultado Esperado

  • Interfaz con dos inputs y 4 botones
  • Al hacer click en "Sumar", muestra la suma
  • Al hacer click en "Dividir" con 0, muestra mensaje de error
  • El resultado aparece en la página sin recargar
