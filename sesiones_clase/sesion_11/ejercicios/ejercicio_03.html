Ejercicio 3: To-Do List Interactiva

Sesión 11 - JavaScript III

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea una lista de tareas interactiva (To-Do List) usando funciones y eventos del DOM. La aplicación debe:

  1. Campo de texto para escribir una tarea
  2. Botón para agregar la tarea
  3. Lista que muestre las tareas agregadas
  4. Opción para marcar tarea como completada (click)
  5. Opción para eliminar una tarea
  6. Funciones separadas para cada acción

✅ Requisitos

  • Función agregarTarea(texto)
  • Función marcarCompletada(elemento)
  • Función eliminarTarea(elemento)
  • Usar addEventListener para los eventos
  • Crear elementos dinámicamente con createElement
  • AppendChild para agregar a la lista

💡 Pistas

Ver pistas
  • let li = document.createElement("li") para crear elemento
  • li.textContent = texto para el contenido
  • lista.appendChild(li) para agregar
  • elemento.classList.toggle("completada") para completar
  • elemento.remove() para eliminar
  • input.value = "" para limpiar input

🎯 Resultado Esperado

  • Input para escribir tarea + botón "Agregar"
  • Al agregar, aparece en la lista
  • Click en tarea la marca como completada (tachado)
  • Botón "X" para eliminar cada tarea
  • Diseño visual claro y organizado
