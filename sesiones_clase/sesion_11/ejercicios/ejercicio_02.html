Ejercicio 2: Galería de Imágenes con Eventos

Sesión 11 - JavaScript III

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea una galería de imágenes interactiva con eventos del DOM. La galería debe:

  1. Mostrar una imagen grande principal
  2. Mostrar miniaturas debajo
  3. Al hacer click en una miniatura, cambiar la imagen grande
  4. Al pasar el mouse sobre una miniatura, resaltar visualmente
  5. Usar placeholder images (ej. via.placeholder.com o similar)

✅ Requisitos

  • Usar querySelectorAll para las miniaturas
  • Usar addEventListener para click
  • Usar eventos mouseover/mouseout para efecto hover
  • Cambiar el atributo src de la imagen principal
  • Al menos 4 imágenes en la galería

💡 Pistas

Ver pistas
  • imagenGrande.src = this.src para cambiar imagen
  • this.classList.add("activa") para destacar
  • Usa imágenes de placeholder como: https://via.placeholder.com/600x400
  • Miniaturas: https://via.placeholder.com/150x100

🎯 Resultado Esperado

  • Imagen principal grande arriba
  • 4 miniaturas debajo
  • Al hacer click en miniatura, cambia la imagen grande
  • Las miniaturas tienen efecto visual al pasar el mouse
