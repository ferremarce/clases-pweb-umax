Sesión 11: JavaScript III - Funciones y Eventos del DOM

Ejercicios de Práctica

Prof. Ing. Juan M. Ferreira

Instrucciones Generales

En esta sesión practicarás la creación de funciones en JavaScript y la manipulación de eventos del DOM para crear interactividad en páginas web.

Duración: 2.5 horas

Nota: Usa funciones para organizar el código y eventos para responder a las acciones del usuario.
