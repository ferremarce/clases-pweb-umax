¿Qué son las Funciones?

Las funciones son bloques de código reutilizables que realizan una tarea específica.


  • Evitan código repetido
  • Facilitan el mantenimiento
  • Permiten modularizar
  • Pueden recibir parámetros
  • Pueden retornar valores

Declaración de Funciones

// Función tradicional (declaración)
function saludar(nombre) {
    return "Hola, " + nombre;
}

// Llamar a la función
let mensaje = saludar("Juan");
alert(mensaje); // "Hola, Juan"

Expresión de Función

// Expresión de función
let saludar = function(nombre) {
    return "Hola, " + nombre;
};

// Arrow function (más moderno)
let saludar = (nombre) => "Hola, " + nombre;

// Arrow function con cuerpo
let saludar = (nombre) => {
    return "Hola, " + nombre;
};
💡 Ventaja de arrow functions: Sintaxis más corta y contexto léxico de this.

Parámetros y Retorno

// Función con múltiples parámetros
function sumar(a, b) {
    return a + b;
}

// Función con valor por defecto
function saludar(nombre = "Usuario") {
    return "Hola, " + nombre;
}

// Función sin retorno
function mostrarMensaje(msg) {
    alert(msg);
}

  • return devuelve un valor
  • Parámetros pueden tener valores por defecto

El DOM (Document Object Model)

El DOM es la representación en memoria del documento HTML como objetos JavaScript.


  • document: objeto principal
  • Cada etiqueta HTML es un elemento
  • JavaScript puede modificarlo
  • Árbol jerárquico
🎯 Pruébalo: Abre la consola y escribe: document.body

Seleccionar Elementos

// Por ID (más rápido)
let elemento = document.getElementById("miId");

// Por selector CSS (más flexible)
let elemento = document.querySelector(".miClase");
let elementos = document.querySelectorAll(".miClase");

// Por tag
let elementos = document.getElementsByTagName("p");
⚠️ Importante: Si el elemento no existe, getElementById retorna null.

Eventos del DOM

Los eventos son acciones que ocurren en la página.


Evento Descripción
click Cuando se hace click
mouseover Cuando el mouse entra
mouseout Cuando el mouse sale
submit Al enviar formulario
input Al escribir en input

addEventListener

// Seleccionar elemento
let boton = document.getElementById("miBoton");

// Agregar evento
boton.addEventListener("click", function() {
    alert("¡Hiciste click!");
});

// Con arrow function
boton.addEventListener("click", () => {
    alert("¡Hiciste click!");
});

Modificar Contenido

//Cambiar texto interno
elemento.textContent = "Nuevo texto";
elemento.innerHTML = "<strong>Negrilla</strong>";

// Obtener valor de input
let valor = inputElement.value;

// Establecer valor
inputElement.value = "Nuevo valor";

Modificar Estilos

// Cambiar estilo directamente
elemento.style.color = "red";
elemento.style.backgroundColor = "yellow";
elemento.style.display = "none";

// Cambiar clase
elemento.classList.add("activa");
elemento.classList.remove("oculta");
elemento.classList.toggle("visible");

Ejemplo: Cambiar Color

<button id="cambiarColor">Cambiar Color</button>
<p id="texto">Este texto cambiará de color</p>

<script>
let boton = document.getElementById("cambiarColor");
let texto = document.getElementById("texto");

boton.addEventListener("click", () => {
    // Color aleatorio
    let color = "#" + Math.floor(Math.random()*16777215).toString(16);
    texto.style.color = color;
});
</script>

Ejemplo: Mostrar/Ocultar

<button id="toggle">Mostrar/Ocultar</button>
<div id="contenido" class="oculto">Contenido</div>

<script>
let boton = document.getElementById("toggle");
let contenido = document.getElementById("contenido");

boton.addEventListener("click", () => {
    contenido.classList.toggle("oculto");
});
</script>
