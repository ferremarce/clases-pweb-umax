Ejercicio 1: Tabla de Datos

Sesión 4 - HTML III: Tablas y Formularios

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea una tabla HTML con estructura semántica que muestre las calificaciones de estudiantes:

  • Usa las etiquetas semánticas: thead, tbody, tfoot
  • Encabezados con th (nombre, materia 1, materia 2, promedio)
  • Al menos 5 estudiantes con sus calificaciones
  • En el pie (tfoot) muestra el promedio general

✅ Requisitos

  • Debe tener thead con fila de encabezados
  • Debe tener tbody con datos de estudiantes
  • Debe tener tfoot con totales/promedio
  • Usa th para encabezados de columna
  • Usa scope="col" en los th

💡 Pistas

Ver pistas
  • Estructura: <table> → <thead> → <tr> → <th>
  • Promedio se calcula manualmente para el tfoot
  • Usa <th scope="col"> para encabezados

🎯 Resultado Esperado

Una tabla profesional con encabezados, datos de estudiantes y un pie de tabla con el promedio general.
