Ejercicio 2: Formulario de Registro

Sesión 4 - HTML III: Tablas y Formularios

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea un formulario de registro de usuario con diferentes tipos de campos:

  • Usa fieldset para agrupar campos relacionados
  • Campos: nombre, email, contraseña, fecha de nacimiento
  • Un select para seleccionar país
  • Un grupo de radio buttons para género
  • Un checkbox para términos y condiciones

✅ Requisitos

  • Al menos 2 fieldset con legend
  • Labels asociados a cada input (atributo for)
  • type="email" para el campo email
  • type="password" para contraseña
  • type="date" para fecha de nacimiento
  • type="radio" para opciones de género (mismo name)
  • type="checkbox" para términos
  • select con al menos 3 opciones

💡 Pistas

Ver pistas
  • Usa el mismo valor en el atributo name para todos los radio buttons del mismo grupo
  • Estructura fieldset: <fieldset><legend>Título</legend>...</fieldset>
  • Select: <select><option value="py">Paraguay</option></select>

🎯 Resultado Esperado

Un formulario profesional con campos agrupados visualmente, diferentes tipos de input y lista desplegable.
