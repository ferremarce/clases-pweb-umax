Ejercicio 1: Calculadora con Menú

Sesión 10 - JavaScript II

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea una calculadora interactiva con menú usando estructuras condicionales if/else. El programa debe:

  1. Mostrar un menú con las siguientes opciones:
    • 1 = Sumar dos números
    • 2 = Restar dos números
    • 3 = Multiplicar dos números
    • 4 = Determinar el mayor de dos números
    • 5 = Verificar si un número es par o impar
  2. Según la opción seleccionada, pedir los números necesarios
  3. Realizar la operación correspondiente
  4. Mostrar el resultado con alert()
  5. Manejar opciones inválidas

✅ Requisitos

  • Usar if/else para las decisiones
  • Convertir inputs a número con Number()
  • Validar que los números sean válidos
  • Manejar división por cero en la resta (opcional)
  • Mensaje claro para opción inválida

💡 Pistas

Ver pistas
  • Usa prompt() para el menú y los números
  • Para par/impar: numero % 2 === 0 es par
  • Para el mayor: usa operador ternario o if
  • Usa else if para las diferentes opciones
