Ejercicio 3: Simulación de Notas

Sesión 10 - JavaScript II

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea un sistema de gestión de notas usando switch y for. El sistema debe:

  1. Un array con las notas de 5 estudiantes: [75, 92, 45, 88, 70]
  2. Usar un bucle for para iterar por cada estudiante
  3. Usar switch para determinar la calificación:
    • 90-100: A (Excelente)
    • 80-89: B (Muy Bueno)
    • 70-79: C (Bueno)
    • 60-69: D (Suficiente)
    • 0-59: F (Reprobado)
  4. Mostrar mensaje por cada estudiante
  5. Al final, mostrar el promedio general

✅ Requisitos

  • Usar array de notas
  • Usar for para iterar
  • Usar switch con rangos
  • Calcular promedio
  • Mostrar resultado por estudiante y promedio

💡 Pistas

Ver pistas
  • Para rangos en switch, verifica rango primero: case 90: case 91: ... case 100:
  • O usa múltiples casos juntos: case "90": case "91": ...
  • Para el promedio: suma todas las notas y divide por el array.length
  • Usa toFixed(2) para mostrar el promedio con 2 decimales
