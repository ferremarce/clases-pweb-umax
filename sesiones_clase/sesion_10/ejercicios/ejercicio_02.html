Ejercicio 2: Gestor de Tareas v2

Sesión 10 - JavaScript II

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea un programa que funcione como un gestor de tareas interactivo con menú persistente usando bucles, condicionales y arrays. El programa debe:

  1. Inicializar un array vacío para almacenar las tareas
  2. Mostrar un menú repetitivo con while con las opciones:
    • 1 = Agregar tarea
    • 2 = Ver todas las tareas
    • 3 = Completar tarea
    • 4 = Mostrar resumen
    • 5 = Salir
  3. Implementar cada opción usando condicionales if/else o switch
  4. Usar for para recorrer el array al mostrar tareas
  5. Salir del programa con break

✅ Requisitos

  • Usar un array para almacenar las tareas como objetos con nombre y estado
  • Usar while para el menú principal
  • Usar switch o if/else para procesar cada opción
  • Usar for para iterar sobre las tareas
  • Usar break para salir del bucle
  • Usar prompt() para recibir datos
  • Usar alert() para mostrar resultados

💡 Pistas

Ver pistas
  • Estructura de tarea: { nombre: "Tarea 1", completada: false }
  • Agregar tarea: tareas.push({ nombre: nombre, completada: false })
  • Mostrar tareas: for (let i = 0; i < tareas.length; i++)
  • Para saber pendientes: contar tareas donde completada === false
  • Switch necesita break al final de cada caso
