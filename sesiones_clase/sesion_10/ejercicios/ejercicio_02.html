Ejercicio 2: Juego de Adivinanza

Sesión 10 - JavaScript II

Prof. Ing. Juan M. Ferreira

📝 Enunciado

Crea un juego de adivinanza de número usando un bucle while. El juego debe:

  1. Generar un número aleatorio entre 1 y 100 (usa Math.floor(Math.random() * 100) + 1)
  2. Dar al usuario un máximo de 5 intentos
  3. En cada intento, pedir un número usando prompt()
  4. Indicar si el número secreto es mayor o menor al ingresado
  5. Mostrar mensaje de victoria si adivina, o de derrota si se agotan los intentos

✅ Requisitos

  • Usar bucle while para los intentos
  • Contador de intentos
  • Comparación con if/else para dar pistas
  • Usar break para salir al adivinar
  • Mensaje final con el número secreto

💡 Pistas

Ver pistas
  • Math.random() genera 0-0.999..., multiplica por 100 y suma 1 para 1-100
  • Usa un contador que aumente en cada intento
  • Para comparar: intento < numeroSecreto = "Es mayor"
  • Si adivina, usa break para salir del while

🎯 Resultado Esperado

// Genera número aleatorio: 42

// Prompt: "Adivina el número (1-100). Intentos: 5"
// Input: 50
// Alert: "Es menor, intenta de nuevo"

// Prompt: "Adivina el número (1-100). Intentos: 4"
// Input: 42
// Alert: "¡Felicidades! Adivinaste en X intentos"
