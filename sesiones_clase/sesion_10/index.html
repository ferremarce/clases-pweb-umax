Condicionales: if / else

Los condicionales permiten ejecutar código según una condición.


if (condición) {
    // código si es verdadero
} else {
    // código si es falso
}
Ejemplo: 
let edad = 18;

if (edad >= 18) {
    alert("Eres mayor de edad");
} else {
    alert("Eres menor de edad");
}

Condicionales: else if

Para múltiples condiciones, usar else if.


let nota = 85;

if (nota >= 90) {
    alert("Excelente");
} else if (nota >= 70) {
    alert("Aprobado");
} else if (nota >= 50) {
    alert("Reprobado");
} else {
    alert("Repetición obligatoria");
}

Operador Ternario

Sintaxis reducida para condiciones simples:


condición ? valorSiVerdadero : valorSiFalso

Ejemplo:

let edad = 20;
let mensaje = edad >= 18 ? "Adulto" : "Menor";
alert(mensaje); // "Adulto"

// Equivalente con if/else:
let mensaje2;
if (edad >= 18) {
    mensaje2 = "Adulto";
} else {
    mensaje2 = "Menor";
}

Switch: Estructura

Para comparar una variable contra múltiples valores.


switch (variable) {
    case valor1:
        // código si variable === valor1
        break;
    case valor2:
        // código si variable === valor2
        break;
    default:
        // código si no coincide ningún caso
}
⚠️ Importante: El break es necesario para salir del switch. Sin él, se ejecutan los casos siguientes.

Switch: Ejemplo

let dia = prompt("Ingresa un día (1-7):");

switch (dia) {
    case "1":
        alert("Domingo");
        break;
    case "2":
        alert("Lunes");
        break;
    case "3":
        alert("Martes");
        break;
    case "4":
        alert("Miércoles");
        break;
    case "5":
        alert("Jueves");
        break;
    case "6":
        alert("Viernes");
        break;
    case "7":
        alert("Sábado");
        break;
    default:
        alert("Día inválido");
}

Bucles: while

Repite código mientras la condición sea verdadera.


while (condición) {
    // código a repetir
}

Ejemplo - Contar hasta 5:

let contador = 1;

while (contador <= 5) {
    console.log(contador);
    contador++;
}
// Resultado: 1, 2, 3, 4, 5

Bucles: do-while

Ejecuta el código al menos una vez, luego verifica la condición.


do {
    // código a repetir
} while (condición);

Ejemplo:

let numero;

do {
    numero = prompt("Ingresa un número mayor a 10:");
} while (numero <= 10);

alert("¡Gracias!");
💡 Diferencia: while verifica antes, do-while verifica después (siempre ejecuta al menos una vez).

Bucles: for

Bucle con contador, ideal para repeticiones conocidas.


for (inicialización; condición; incremento) {
    // código a repetir
}

Ejemplo - Contar hasta 5:

for (let i = 1; i <= 5; i++) {
    console.log(i);
}
// Resultado: 1, 2, 3, 4, 5

Break y Continue

Palabra Función Ejemplo
break Sale del bucle completamente 
for (...) {
    if (error) break;
}
continue Salta a la siguiente iteración 
for (...) {
    if (par) continue;
}

Ejemplo: Tabla de Multiplicar

// Tabla del 5 usando for
for (let i = 1; i <= 10; i++) {
    console.log("5 x " + i + " = " + (5 * i));
}

// Resultado:
// 5 x 1 = 5
// 5 x 2 = 10
// ...
// 5 x 10 = 50

Ejemplo: Validación de Contraseña

let intentos = 0;
const MAX_INTENTOS = 3;
let contraseñaCorrecta = "secreto123";

while (intentos < MAX_INTENTOS) {
    let intento = prompt("Ingresa la contraseña:");
    
    if (intento === contraseñaCorrecta) {
        alert("¡Acceso concedido!");
        break;
    } else {
        intentos++;
        alert("Contraseña incorrecta. Intentos: " + intentos);
    }
}

if (intentos === MAX_INTENTOS) {
    alert("Usuario bloqueado");
}

Iterar Arrays con for

// Array de nombres
let nombres = ["Ana", "Juan", "Pedro", "María"];

// Recorrer con for clásico
for (let i = 0; i < nombres.length; i++) {
    console.log("Hola, " + nombres[i]);
}

// Resultado:
// Hola, Ana
// Hola, Juan
// Hola, Pedro
// Hola, María
💡Nota: .length devuelve la cantidad de elementos del array.
