Solución: Ejercicio 1 - Box-Sizing

Comparación de Box-Sizing

Ambas cajas tienen: width: 150px, padding: 20px, border: 5px

content-box
→ 200px total
border-box
→ 150px total

Código Completo

/* CSS - Archivo externo */
* {
    box-sizing: border-box;
}

.caja {
    width: 150px;
    padding: 20px;
    border: 5px solid #8B0000;
}

.content-box {
    box-sizing: content-box; /* Por defecto */
}

.border-box {
    box-sizing: border-box; /* Recomendado */
}

Explicación

  • content-box: El width (150px) solo incluye el contenido. El padding y border se suman: 150 + 40 + 10 = 200px
  • border-box: El width (150px) incluye contenido + padding + border. El elemento siempre mide 150px

Resultado Visual

content-box
Ancho real: 200px
border-box
Ancho real: 150px

💡 border-box facilita los cálculos y es el estándar recomendado
