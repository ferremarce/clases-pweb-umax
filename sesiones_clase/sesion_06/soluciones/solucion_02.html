Solución: Ejercicio 2 - Posicionamiento

Código CSS

.tarjeta-producto {
    position: relative;
    overflow: hidden;
}

.badge {
    position: absolute;
}

.badge-descuento {
    top: 15px;
    right: -30px;
    transform: rotate(45deg);
}

.badge-nuevo {
    top: 10px;
    left: 10px;
    border-radius: 20px;
}

.badge-oferta {
    top: 10px;
    right: 10px;
    border-radius: 50%;
}

HTML

<article class="tarjeta-producto">
    <span class="badge badge-descuento">-20%</span>
    <div class="imagen-producto">Imagen</div>
    <h3>Producto</h3>
    <p>Descripción del producto</p>
    <span class="precio">$79.99</span>
</article>

Explicación

  • position: relative en el contenedor establece un contexto de posicionamiento
  • position: absolute permite posicionar el elemento respecto al contenedor padre
  • top, right, left, bottom definen la posición exacta
  • transform: rotate() rota el elemento para efectos decorativos
  • overflow: hidden recorta el badge si se sale del contenedor
