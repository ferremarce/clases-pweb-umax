Solución: Ejercicio 3 - Header Fijo y Overflow

CSS: Header Fijo

* {
    box-sizing: border-box;
}

body {
    margin: 0;
}

header {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    z-index: 1000;
    background: linear-gradient(135deg, #8B0000, #DC143C);
    color: white;
    padding: 15px 20px;
}

main {
    margin-top: 80px;
    padding: 20px;
}

Overflow

Controla cómo se muestra el contenido que se desborda. Usa 3 contenedores en la misma línea:

overflow: auto

CSS: Footer Fijo

footer {
    position: fixed;
    bottom: 0;
    left: 0;
    width: 100%;
    z-index: 1000;
    background: linear-gradient(135deg, #8B0000, #DC143C);
    color: white;
    padding: 15px 20px;
}

body {
    padding-bottom: 70px;
}

Puntos Clave

  • position: fixed posiciona respecto a la ventana del navegador
  • z-index asegura que header y footer estén siempre visibles
  • margin-top en main compensa la altura del header
  • padding-bottom en body compensa la altura del footer
  • overflow: auto añade scroll solo cuando es necesario
  • overflow: hidden recorta el contenido que sobresale
  • overflow: visible es el comportamiento por defecto
  • display: flex alinea los contenedores en línea
