Solución: Ejercicio 3 - Header Fijo y Overflow

CSS: Header Fijo

* {
    box-sizing: border-box;
}

body {
    margin: 0;
}

header {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    z-index: 1000;
    background: linear-gradient(135deg, #8B0000, #DC143C);
    color: white;
    padding: 15px 20px;
}

main {
    margin-top: 80px;
    padding: 20px;
}

HTML

<header>
    <h1>Mi Sitio Web</h1>
    <nav>
        <a href="#">Inicio</a>
        <a href="#">Acerca</a>
        <a href="#">Contacto</a>
    </nav>
</header>

<main>
    <section>...</section>
</main>

Overflow

Controla cómo se muestra el contenido que se desborda:

overflow: auto

Este contenido es muy largo y necesita scroll para verse completo. Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam.

overflow: hidden

Este contenido está oculto y no se puede ver completamente. Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

Bonus: Float con Clearfix

.contenedor::after {

.imagen-flotante {

Puntos Clave

  • position: fixed posiciona respecto a la ventana del navegador
  • z-index asegura que el header esté siempre visible
  • margin-top en main compensa la altura del header
  • overflow: auto añade scroll solo cuando es necesario
  • overflow: hidden recorta el contenido que sobresale
