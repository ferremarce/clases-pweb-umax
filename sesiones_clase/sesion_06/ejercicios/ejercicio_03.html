Ejercicio 3: Header Fijo y Overflow

Enunciado

Mejora tu página personal añadiendo un header fijo y controlando el overflow de contenedores.

Parte 1: Header Fijo

  • Crea un header con position: fixed
  • Usa top: 0 y width: 100%
  • Aplica z-index: 1000 para que esté siempre arriba
  • Agrega margin-top al body o contenedor principal para compensar

Parte 2: Overflow con Flex

Crea 3 contenedores en la misma línea que demuestren los diferentes valores de overflow:

  • Usa display: flex o display: inline-flex en el contenedor padre
  • overflow: auto - Añade scroll solo cuando el contenido desborda
  • overflow: hidden - Recorta el contenido que sobresale
  • overflow: visible - Comportamiento por defecto, el contenido se desborda

Usa una altura fija (ej: 80px) para cada contenedor y contenido largo para ver el efecto.

Parte 3: Footer Fijo

Ahora añade un footer que permanezca fijo en la parte inferior de la pantalla:

  • Crea un <footer> con position: fixed
  • Usa bottom: 0 para posicionarlo en la parte inferior
  • Aplica width: 100% y z-index: 1000
  • Agrega padding-bottom al body o contenedor principal para evitar que el contenido quede oculto

CSS: Header Fijo

header {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    z-index: 1000;
    background: linear-gradient(135deg, #8B0000, #DC143C);
    color: white;
    padding: 15px 20px;
}

main {
    margin-top: 80px;
}

CSS: Overflow (3 contenedores en línea)

.contenedor-overflow {
    display: flex;
    gap: 20px;
}

.caja-overflow {
    flex: 1;
    min-width: 200px;
    height: 100px;
    border: 2px solid #8B0000;
}

/* Scroll automático cuando es necesario */
.contenedor-scroll {
    overflow: auto;
}

/* Recorta el contenido que sobresale */
.recortar-contenido {
    overflow: hidden;
}

/* Comportamiento por defecto */
.desbordar {
    overflow: visible;
}
