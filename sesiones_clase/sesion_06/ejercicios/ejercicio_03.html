Ejercicio 3: Header Fijo y Overflow

Enunciado

Mejora tu página personal añadiendo un header fijo y controlando el overflow de contenedores.

Parte 1: Header Fijo

  • Crea un header con position: fixed
  • Usa top: 0 y width: 100%
  • Aplica z-index: 1000 para que esté siempre arriba
  • Agrega margin-top al body o contenedor principal para compensar

Parte 2: Overflow

  • Usa overflow: auto en contenedores con mucho contenido
  • Usa overflow: hidden para recortar contenido excesivo

CSS: Header Fijo

header {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    z-index: 1000;
    background: linear-gradient(135deg, #8B0000, #DC143C);
    color: white;
    padding: 15px 20px;
}

main {
    margin-top: 80px;
}

CSS: Overflow

.contenedor-scroll {
    height: 200px;
    overflow: auto;
}

.recortar {
    height: 100px;
    overflow: hidden;
}
