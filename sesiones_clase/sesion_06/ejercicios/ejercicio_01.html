Ejercicio 1: Modelo de Caja y Box-Sizing

Enunciado

Crea una página que demuestre la diferencia entre box-sizing: content-box (por defecto) y box-sizing: border-box.

Requisitos:

  • Aplicabox-sizing: border-box global con el selector universal (*)
  • Crea 4 cajas con las mismas medidas: width: 150px, padding: 20px, border: 5px
  • Observa cómo el ancho total varía según el box-sizing

HTML Sugerido

<div class="medidas">
    <div class="medida-caja sin-boxsizing">
        <strong>content-box</strong>
        <p>width: 150px</p>
        <p>padding: 20px</p>
        <p>border: 5px</p>
        <p>Total: 200px</p>
    </div>
    <div class="medida-caja con-boxsizing">
        <strong>border-box</strong>
        <p>width: 150px</p>
        <p>padding: 20px</p>
        <p>border: 5px</p>
        <p>Total: 150px</p>
    </div>
</div>

CSS Sugerido

.medida-caja {
    width: 150px;
    padding: 20px;
    border: 5px solid #8B0000;
}

.sin-boxsizing {
    box-sizing: content-box;
}

.con-boxsizing {
    box-sizing: border-box;
}

Resultado Esperado

Dos cajas, una con content-box (ancho 200px visual) y otra con border-box (ancho 150px visual).

padding: 20px

border: 5px

Total: 200px

border: 5px
