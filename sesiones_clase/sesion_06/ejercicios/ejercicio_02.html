Ejercicio 2: Posicionamiento Relative y Absolute

Enunciado

Crea tarjetas de productos con badges de oferta posicionados absolutamente.

Requisitos:

  • El contenedor de la tarjeta debe tener position: relative
  • El badge debe tener position: absolute
  • Usa top/right/left/bottom para posicionar el badge
  • Crea al menos 2 tarjetas diferentes

HTML Sugerido

<article class="tarjeta-producto">
    <span class="badge">20% DESC</span>
    <img src="imagen.jpg" alt="Producto">
    <h3>Producto</h3>
    <p>Descripción</p>
    <span class="precio">$99.99</span>
</article>

CSS Sugerido

.tarjeta-producto {
    position: relative;
    overflow: hidden;
}

.badge {
    position: absolute;
    top: 15px;
    right: -30px;
    transform: rotate(45deg);
}

Variación: Badge Esquina

.badge-oferta {
    position: absolute;
    top: 10px;
    left: 10px;
    border-radius: 20px;
}
