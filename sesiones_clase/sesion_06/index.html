Box-Sizing: El Problema

El modelo de caja por defecto incluye el padding y border dentro del width:

content-box
width = 200 + 40 + 10 = 250px
border-box
width = 200px (incluye padding)
⚠️ Problema: Con content-box, al agregar padding/border, el elemento crece más allá del width esperado.

La Solución: box-sizing

La propiedad box-sizing cambia cómo se calcula el tamaño:

content-box (por defecto)

box-sizing: content-box;
width: 200px;
padding: 20px;
border: 5px solid black;
/* Ancho total: 200 + 40 + 10 = 250px */

border-box (recomendado)

box-sizing: border-box;
width: 200px;
padding: 20px;
border: 5px solid black;
/* Ancho total: 200px (el padding está dentro) */
💡 Mejor práctica: Usar * { box-sizing: border-box; } al inicio de todo CSS.

La Propiedad Display

Display define cómo se comporta un elemento en el flujo de la página:

block

Ocupa todo el ancho disponible, comienza en nueva línea:

display: block;
Elemento block 1
Elemento block 2

inline

Comparte línea con otros elementos, ignora width/height:

display: inline;
Texto inline 1 Texto inline 2

Display: inline-block

Combina las ventajas de ambos:

inline-block

Comparte línea + acepta width/height:

display: inline-block;
Botón 1 Botón 2 Botón 3

none

Oculta el elemento (no ocupa espacio):

display: none;
/* Oculta y remove del flujo */

Posicionamiento en CSS

La propiedad position determina el método de posicionamiento:

static

Por defecto, sigue el flujo normal

position: static;

relative

Posición relativa a su posición original

position: relative;

absolute

Se posiciona respecto al ancestro posicionado

position: absolute;

fixed

Fijo respecto a la ventana del navegador

position: fixed;

Position: Relative

El elemento se mueve respecto a su posición original:

Ejemplo

.caja {
    position: relative;
    top: 20px;
    left: 30px;
}
Me moví 20px abajo y 30px derecha
  • top, right, bottom, left: desplazan respecto a posición original
  • El espacio original se preserva en el flujo

Position: Absolute

El elemento se posiciona respecto al ancestro posicionado más cercano:

Ejemplo

.tarjeta { position: relative; }

.badge {
    position: absolute;
    top: 10px;
    right: 10px;
}

Características:

  • Se remueve del flujo normal
  • El ancesto debe tener position: relative|absolute|fixed
  • Si no hay ancesto, se posiciona respecto al body

Position: Fixed

El elemento se posiciona respecto a la ventana del navegador:

Ejemplo común: Menú fijo

header {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    z-index: 1000;
}
  • Se mantiene visible al hacer scroll
  • No importa el ancesto
  • También se remove del flujo

Z-Index: Control de Capas

z-index controla qué elemento aparece encima:

Ejemplo

.caja-1 {
    position: absolute;
    z-index: 1;
}

.caja-2 {
    position: absolute;
    z-index: 2;
}
z-index: 1
z-index: 2
  • Mayor z-index = aparece encima
  • Solo funciona con position != static

Overflow

Controla qué sucede cuando el contenido sobrepasa su contenedor:

overflow: visible (default)

Este contenido es muy largo y se desborda fuera del contenedor.

overflow: hidden

Este contenido es muy largo y se recorta.

overflow: auto

Este contenido es muy largo y tiene scroll.

Float: Layouts Legacy

Float hace que el elemento flote a un lado, permitiendo que el texto fluya alrededor:

float: left

El logo flota a la izquierda y el texto fluye alrededor.

Sintaxis

float: left;
float: right;
float: none;
💡 Nota: Float fue el método principal de layouts antes de Flexbox y Grid. Hoy se usa principalmente para imágenes dentro de texto.

Clearfix: Problema y Solución

Float remove el elemento del flujo, causando problemas con contenedores:

Problema

.contenedor { border: 1px solid black; }

.imagen { float: left; }
/* El contenedor se colapsa */

Solución: Clearfix

.contenedor::after {
    content: "";
    display: block;
    clear: both;
}

Ejemplo Práctico: Tarjeta con Badge

Vamos a crear una tarjeta con una insignia posicionada:

Oferta

Producto Destacado

Descripción del producto con características relevantes.

HTML

<article class="tarjeta">
    <span class="badge">Oferta</span>
    <h3>Producto</h3>
    <p>Descripción</p>
</article>

CSS

.tarjeta { position: relative; }

.badge {
    position: absolute;
    top: -10px;
    right: -10px;
}

Buenas Prácticas

✅ Hacer:

  • Usar box-sizing: border-box global
  • Usar position: relative en el contenedor padre
  • Usar z-index para control de capas
  • Aplicar overflow: auto a contenedores con contenido dinámico
  • Usar float solo para imágenes

❌ Evitar:

  • Olvidar box-sizing: border-box
  • Usar z-index sin position
  • position: absolute para todo el layout
  • float para layouts completos (usar Flexbox/Grid)
