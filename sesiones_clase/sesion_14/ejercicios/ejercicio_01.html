Ejercicio 1: Navbar y Cards

📝 Enunciado

Crea una página web usando componentes de Bootstrap: navbar responsivo y tarjetas de proyectos.

  1. Navbar con logo, menú de navegación y botón hamburguesa
  2. Sección de proyectos con 4 cards
  3. Cada card debe tener imagen, título, descripción y botón

✅ Requisitos

  • Usar navbar-expand-lg con collapse
  • Usar navbar-dark bg-dark
  • Card con imagen, card-body, card-title, card-text
  • btn btn-primary en cada card
  • Usar container-fluid o container

💡 Pistas

Ver pistas
  • <nav class="navbar navbar-expand-lg navbar-dark bg-dark">
  • <div class="card" style="width: 18rem;">
  • card-img-top para la imagen
  • Bootstrap Icons para iconos
