Ejercicio 2: Portafolio con Modal y Formulario

📝 Enunciado

Crea un portafolio web con carrusel, modal de contacto y formulario validado.

  1. Carrusel con 3 imágenes
  2. Sección "Sobre mí" con información personal
  3. Botón que abre un modal de contacto
  4. Formulario dentro del modal con validación

✅ Requisitos

  • carousel con indicators y controls
  • data-bs-toggle="modal" para abrir
  • form-control en inputs
  • was-validated para validación
  • is-valid / is-invalid en inputs

💡 Pistas

Ver pistas
  • <div id="carouselExample" class="carousel slide">
  • data-bs-target="#miModal" para trigger
  • <input class="form-control" required>
  • Agregar was-validated al form
