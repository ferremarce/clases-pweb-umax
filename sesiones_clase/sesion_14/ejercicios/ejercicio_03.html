Ejercicio 3: Landing Page Completa

📝 Enunciado

Crea una landing page completa integrando todos los componentes de Bootstrap.

  1. Navbar completo con logo y menú
  2. Hero section con título y botones
  3. Features con accordion
  4. Galería con carrusel
  5. Footer con iconos sociales

✅ Requisitos

  • Navbar responsivo completo
  • d-flex y justify-content-center para centrar
  • accordion con collapse
  • carousel en galería
  • bi bi-* para iconos sociales
  • bg-* y text-* para colores

💡 Pistas

Ver pistas
  • Usar d-flex para layouts
  • <div class="accordion"> para FAQ
  • <i class="bi bi-facebook"></i> para redes
  • Combinar varias clases de utilidad
